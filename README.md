# Windows 贪吃蛇游戏 🕹️

一个使用 Zen-C 编写的 Windows 贪吃蛇游戏。

## ✨ 项目特色

- **Zen-C 语言**: 使用现代系统编程语言 Zen-C 开发
- **窗体界面**: 精美的 Windows 窗体界面
- **零依赖**: 不需要安装任何运行时环境
- **直接运行**: 编译后生成独立可执行文件

## 🚀 快速开始

### 下载项目

```bash
git clone https://github.com/xingmu/windows-minigames-diy.git
cd windows-minigames-diy
```

### 编译运行

```bash
cd src
build_windows.bat
```

编译完成后，在 build 目录下生成可执行文件，直接双击运行即可。

## 🎮 游戏控制

| 按键 | 功能 |
|------|------|
| ↑ | 向上移动 |
| ↓ | 向下移动 |
| ← | 向左移动 |
| → | 向右移动 |
| 空格 | 暂停/继续 |
| Enter | 重新开始 |

## 🛠️ 技术栈

- **语言**: Zen-C
- **图形**: Win32 API
- **编译器**: GCC / MinGW

## ⚠️ 重要约束

- **平台**: 仅支持 Windows 10/11
- **语言**: 必须使用 Zen-C 语言开发（禁止使用C语言）
