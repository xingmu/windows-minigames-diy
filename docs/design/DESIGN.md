# 038-RacingPlus 设计文档

## 概述

038-RacingPlus 是 024-Racing 的增强版本，增加了多条赛道和更高的游戏难度。

## 设计目标

1. **多赛道系统**：提供5条风格各异的赛道
2. **难度递进**：从简单到极限，适合不同水平的玩家
3. **视觉增强**：粒子效果、速度线、轮胎痕迹
4. **竞赛深度**：时间限制、圈速记录、检查点系统

## 赛道设计

### 赛道1: Sunset Oval (椭圆赛道)
- **难度**: Easy
- **特点**: 宽赛道，无障碍，适合新手入门
- **圈数**: 3圈
- **设计**: 标准椭圆形，两端弧形

### 赛道2: Technical Twist (技术赛道)
- **难度**: Normal
- **特点**: L形路线，需要精确转向
- **圈数**: 3圈
- **障碍物**: 2个
- **设计**: 直道+弯道组合

### 赛道3: Figure-8 Fury (8字赛道)
- **难度**: Hard
- **特点**: 交叉路口，需要注意路线选择
- **圈数**: 3圈
- **时间限制**: 120秒
- **障碍物**: 4个
- **设计**: 8字形，两条赛道交叉

### 赛道4: Karting Chaos (卡丁车道)
- **难度**: Hard
- **特点**: S弯道多，需要连续转向
- **圈数**: 4圈
- **时间限制**: 150秒
- **障碍物**: 5个
- **设计**: 双S弯

### 赛道5: Speedway Extreme (高速赛道)
- **难度**: Extreme
- **特点**: 长直道多，障碍密集
- **圈数**: 5圈
- **时间限制**: 180秒
- **障碍物**: 9个
- **设计**: 高速环形

## 物理系统

### 速度模型
```
加速度 = 4
最大速度 = 10
摩擦系数(赛道) = 0.97
摩擦系数(草地) = 0.88
```

### 转向模型
- 转向效率与当前速度成正比
- 高速时转向效率降低 (速度>80%时效率*0.7)
- 速度为0时无法转向

### 碰撞模型
- 边界碰撞：速度衰减70%，位置反弹
- 障碍物碰撞：速度反向衰减，速度*(-0.3)

## 粒子系统

### 轮胎痕迹
- 条件：赛道外 + 速度 > 2
- 生命周期：120帧
- 颜色：深灰色，渐变透明

### 尘土粒子
- 条件：赛道外 + 速度 > 1
- 生命周期：30-50帧
- 初速度：随机方向
- 颜色：泥土色

### 速度线
- 条件：速度 > 最大速度 * 0.8
- 生命周期：渐隐
- 绘制位置：车尾

## UI系统

### 菜单界面
- 赛道选择（上下键）
- 显示赛道名、难度、圈数、时间限制
- 高亮当前选择

### HUD
- 速度表（动态颜色）
- 圈数显示
- 最佳圈速
- 当前时间
- 时间限制（低于30秒变红）
- 赛道名
- OFF TRACK警告
- 障碍物警告

### 结束画面
- 半透明遮罩
- 完成/超时信息
- 圈数、时间统计
- 最佳圈速
- 返回菜单提示

## 控制方案

| 按键 | 功能 | 状态 |
|------|------|------|
| UP | 加速 | 按住生效 |
| LEFT | 左转 | 按住生效 |
| RIGHT | 右转 | 按住生效 |
| SPACE | 刹车 | 按住生效 |
| ESC | 退出/菜单 | 单次 |
| ENTER | 开始 | 单次 |
| UP/DOWN | 选择赛道 | 菜单中 |

- 速度线数量上限控制
