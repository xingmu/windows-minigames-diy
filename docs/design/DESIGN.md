# 超级玛丽风格游戏设计文档

## 概述

本游戏是一款经典的横版平台跳跃游戏，复刻了超级玛丽的经典玩法。

## 游戏设计

### 核心玩法

1. **移动**: 使用左右方向键控制角色水平移动
2. **跳跃**: 空格键或上方向键跳跃
3. **踩敌人**: 从上方踩到敌人可以消灭它们
4. **收集金币**: 接触金币即可收集

### 游戏元素

#### 玩家角色
- 尺寸: 24x32 像素
- 颜色: 红色帽子/衣服，蓝色裤子
- 动画: 行走、跳跃

#### 敌人 (Goomba)
- 尺寸: 24x24 像素
- 外观: 棕色蘑菇形
- 行为: 左右巡逻

#### 金币
- 尺寸: 16x16 像素
- 外观: 金色圆形，旋转动画
- 效果: +100分

#### 平台
- 草地平台: 绿色草皮 + 棕色泥土
- 城堡: 砖块结构

### 关卡设计

- 总长度: 约1200像素
- 起点: 左上角
- 终点: 右侧城堡
- 平台分布: 错落有致，需要跳跃技巧

## 技术实现

### 框架
- 基于 003-framework
- 直接帧缓冲渲染
- 60 FPS 游戏循环

### 碰撞检测
- AABB 矩形碰撞
- 平台单面碰撞 (只检测上方)
- 敌人可从上方踩死

### 摄像机
- 水平跟随玩家
- 玩家位于画面1/3处

## 分数系统

| 动作 | 分数 |
|------|------|
| 踩死敌人 | +100 |
| 收集金币 | +100 |
| 吃满5金币 | +1生命 |

## 状态机

```
Ready -> (按空格) -> Playing
Playing -> (死亡) -> Ready (重试)
Playing -> (通关) -> Won
Playing/Won -> (按空格) -> Ready
```

## 颜色方案

- 天空: 渐变蓝 (#6B8CFF -> #87CEEB)
- 地面: 棕色 (#8B4513) + 绿色草顶 (#009600)
- 马里奥: 红 (#DC3232) + 蓝 (#3264DC)
- 金币: 金色 (#FFD200)
- 敌人: 棕色 (#965A32)
