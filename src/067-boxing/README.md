# Zen-C Boxing（拳击）游戏

基于 Zen-C 游戏框架的小游戏项目。

## 🎮 游戏简介

一款复古风格的拳击游戏，玩家控制红色拳手与蓝色AI对手进行对战。

## 🎯 游戏特色

- **真实拳击动作**: 刺拳(Jab)和勾拳(Hook)两种攻击方式
- **回合制比赛**: 三回合制，每回合99秒
- **智能AI对手**: AI会根据距离和情况自动调整策略
- **命中判定**: 攻击必须在范围内才能造成伤害
- **血量系统**: 实时显示双方血量

## 🎮 控制说明

| 按键 | 功能 |
|------|------|
| W | 向上移动 |
| A | 向左移动 |
| S | 向下移动 |
| D | 向右移动 |
| J / 空格 | 刺拳（快速，伤害5） |
| K / 回车 | 勾拳（较慢，伤害12） |

## 🔧 构建说明

### Zen-C 编译器

需要安装 Zen-C 编译器 `zc`。

### Linux/macOS

```bash
make all      # 编译游戏
make run      # 编译并运行
make clean    # 清理构建文件
```

### Windows

```bash
build_windows.bat
```

## 🎯 游戏规则

1. **回合制度**: 游戏采用三回合制
2. **每回合时间**: 每回合99秒
3. **胜利条件**:
   - 对手血量归零（KO胜）
   - 回合时间结束时血量领先
4. **攻击机制**:
   - 刺拳(Jab): 快速，冷却短，伤害5
   - 勾拳(Hook): 较慢，冷却长，伤害12
   - 必须在攻击范围内才能命中
