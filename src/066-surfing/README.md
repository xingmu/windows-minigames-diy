# 066-Surfing 冲浪游戏

基于 Zen-C + 003-framework 开发的冲浪游戏。

## 游戏玩法

- **← → ↑ ↓** 方向键控制冲浪者移动
- **空格** 开始游戏 / 暂停
- **R** 重新开始（游戏结束后）
- **ESC** 退出游戏

## 游戏目标

- 控制冲浪者在海浪中前进
- 躲避障碍物：礁石、浮标、海胆
- 收集物品获得额外分数：海星 (+50)、贝壳 (+30)
- 尽可能获得更高分数

## 构建说明

### Zen-C 编译器

需要安装 Zen-C 编译器 `zc`。

### 构建命令

```bash
# 编译游戏
make all

# 运行游戏
make run

# Windows构建
make windows

# 清理
make clean
```

## 游戏特性

- 流畅的上下左右移动控制
- 五种障碍物/收集物：礁石、浮标、海胆、海星、贝壳
- 渐进式加速系统
- 浪花视觉效果
- 分数、生命值和速度显示
- 无敌时间闪烁保护

## 游戏设计

### 画面设计
- 渐变天空和海洋背景
- 动态波浪效果
- 冲浪者动画帧

### 游戏机制
- 初始3条生命
- 碰撞后获得短暂无敌时间
- 速度随时间逐渐增加
- 障碍物从屏幕上方生成
