# 097-Memory 记忆翻牌游戏

Zen-C 语言开发的增强版记忆翻牌游戏。

## 游戏规则

- **目标**: 在最少的步数内，找出所有成对的相同图案
- **操作**: 点击卡牌翻转，查看图案
- **胜利条件**: 找到所有配对

## 操作说明

| 操作 | 功能 |
|------|------|
| 鼠标点击 | 翻转卡牌 |
| H | 显示提示(-10分) |
| R | 重新开始游戏 |
| ESC | 退出游戏 |

## 游戏特性

- **5×4 卡牌网格** (共 20 张 = 10 对)
- **10 种不同图案** (比标准版多)
- **提示系统** - 按 H 键可显示一对卡牌位置(扣10分)
- **分数奖励** - 不用提示配对成功 +100分，使用提示 +50分
- **视觉效果** - 已配对卡牌带光晕，提示卡牌闪烁
- **胜利统计** - 显示平均步数/对

## 与 059-Memory 的区别

| 特性 | 059-Memory | 097-Memory |
|------|------------|------------|
| 网格 | 4×4 (16张) | 5×4 (20张) |
| 对数 | 8 对 | 10 对 |
| 图案 | 8 种 | 10 种 |
| 提示 | 无 | H键提示(-10分) |
| 计分 | 固定+100 | 提示影响得分 |

## 开发

游戏使用帧缓冲 (Framebuffer) 进行渲染，直接写入显存实现高效绘制。
