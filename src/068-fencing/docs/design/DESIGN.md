# 068-Fencing 游戏设计文档

## 1. 概述

- **游戏名称**: Fencing (击剑)
- **类型**: 体育对战游戏
- **目标用户**: Windows 小游戏爱好者
- **核心玩法**: 玩家与 CPU 在剑道上进行击剑对决，先得 5 分获胜

## 2. 游戏设计

### 2.1 视觉风格

- **整体风格**: 简洁优雅的 2D 俯视/侧面视图
- **视角**: 侧面视角 (side-view)，剑道为水平方向
- **配色**:
  - 背景: 深色渐变 (夜空感)
  - 剑道: 木质纹理，标准白色边界线
  - 玩家: 蓝色系
  - CPU: 红色系
  - 命中特效: 金色星爆

### 2.2 角色设计

两个击剑运动员在剑道上对战，每个角色包含:
- 身体 (躯干)
- 头部 (带面罩)
- 双腿
- 双臂
- 击剑剑

### 2.3 动作状态机

```
Idle <-> Attacking <-> Idle
  |          |
  v          v
Parrying  Lunging
  |
  v
  Hit
```

| 状态 | 描述 | 持续时间 |
|------|------|----------|
| Idle | 待机/移动 | - |
| Attacking | 刺剑攻击 | 18 帧 |
| Lunging | 弓步 (大幅前冲) | 30 帧 |
| Parrying | 防守格挡 | 25 帧 |
| Hit | 被击中 | 20 帧 |
| Celebrating | 得分庆祝 | 60 帧 |

## 3. 交互设计

### 3.1 控制方案

| 按键 | 动作 |
|------|------|
| A / ← | 后退 |
| D / → | 前进 |
| J | 刺剑 (快速攻击) |
| K | 弓步 (大幅前冲攻击) |
| S / ↓ | 防守 (格挡) |
| R | 重新开始 |
| Space | 开始回合 |

### 3.2 攻击与防守

- **刺剑 (J)**: 攻击范围 100px，持续 18 帧，冷却 15 帧
- **弓步 (K)**: 攻击范围 140px，持续 30 帧，冷却 20 帧
- **防守 (S)**: 无攻击，但完全格挡敌方攻击
- **同时击中**: 双方均不得分

## 4. 游戏流程

1. **开始界面**: 显示标题和"按 J 开始"
2. **回合开始**: 双方回到各自起始位置
3. **战斗**: 玩家和 CPU 自由对战
4. **得分**: 击中者得分，暂停 1.5 秒
5. **新回合**: 重复步骤 2-4
6. **比赛结束**: 先得 5 分者获胜

## 5. AI 设计

CPU AI 具有以下行为:
- **距离判断**: 保持在攻击范围内
- **攻击决策**: 随机选择刺剑/弓步/防守
- **反应延迟**: 150ms 决策间隔
- **攻击延迟**: 600-1000ms 攻击间隔
- **随机躲避**: 偶尔进行防守
