# Zen-C 迷宫游戏

基于 Zen-C + 003-framework 构建的经典迷宫游戏。

## 游戏特性

- 🎲 **随机迷宫生成** - 每次游戏都是全新的迷宫
- 🧭 **递归回溯算法** - 使用经典算法确保迷宫有解
- 🎯 **目标明确** - 从绿色起点找到红色终点
- 📊 **计分系统** - 步数越少分数越高
- 🎨 **简洁美观** - Zen-C 优雅实现

## 游戏控制

| 按键 | 功能 |
|------|------|
| ↑ / W | 向上移动 |
| ↓ / S | 向下移动 |
| ← / A | 向左移动 |
| → / D | 向右移动 |
| R | 重新开始 |
| ESC | 退出游戏 |

## 游戏规则

1. 玩家从左上角的黄色方块（起点）开始
2. 找到右下角的红色方块（终点）
3. 步数越少分数越高（初始分数1000，每步-2）
4. 迷宫随机生成，每次都不同

## 技术实现

### 迷宫生成算法

使用**递归回溯法**（Recursive Backtracking）生成迷宫：

1. 从起点开始，标记为已访问
2. 随机选择一个方向
3. 如果相邻格子未访问，打通墙壁，递归处理
4. 如果所有方向都已访问，回溯
5. 直到所有格子都被访问

### 数据结构

- `Cell` - 迷宫单元格，包含墙壁信息和访问状态
- `Player` - 玩家位置
- `GameState` - 游戏全局状态
