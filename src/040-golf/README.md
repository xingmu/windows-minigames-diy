# 040-Golf - Zen-C 高尔夫游戏

## 🎯 游戏简介

使用 Zen-C + 003-framework 构建的迷你高尔夫游戏。鼠标蓄力方向，挥杆击球，体验9洞精彩挑战！

## 🎮 游戏特性

- **鼠标蓄力击球** - 点击并拖拽设定方向和力度，松开发射
- **9个特色球洞** - 每洞都有独特的障碍布局
- **物理引擎** - 真实的球运动、摩擦力、墙壁反弹
- **多种障碍物** - 墙壁、水池、沙坑
- **计分系统** - 标准杆、当前杆数、累计成绩

## 🕹️ 操作说明

| 操作 | 说明 |
|------|------|
| 鼠标点击+拖拽 | 瞄准击球方向，拖拽距离决定力度 |
| 松开鼠标 | 发射球 |
| 空格键 | 轻击（小幅力度） |
| 回车键 | 进球后进入下一洞 |
| ESC | 退出游戏 |

## 🎨 障碍物说明

| 颜色 | 类型 | 效果 |
|------|------|------|
| 棕色方块 | 墙壁 | 球反弹 |
| 蓝色区域 | 水障碍 | 罚一杆 |
| 浅黄色区域 | 沙坑 | 摩擦力增大 |

```bash
# 从 src 目录
cd windows-minigames-diy/src/040-golf

# 编译 (需要 Zen-C 编译器)
zcc build -o golf.exe src/main_window.zc

# 运行
./golf.exe
```

## 📝 开发说明

游戏使用 Zen-C 语言开发，基于 003-framework 游戏框架。核心逻辑包括：

- **物理系统** - 球的速度、加速度、摩擦力计算
- **碰撞检测** - 墙壁反弹、障碍物碰撞、边界检测
- **瞄准系统** - 鼠标拖拽方向和距离映射为击球参数
- **状态机** - Ready、Aiming、Dragging、Moving、HoleIn 等状态转换

## 🎯 球洞预览

1. **练习场** - 简单直洞，热身用
2. **L弯道** - L形球道，需要精准控制
3. **水池障碍** - 绕过中心水池
4. **曲折迷宫** - 多重折墙考验
5. **狭长通道** - 在窄道中穿行
6. **圆形迷宫** - 圆形石头障碍
7. **沙坑阵** - 多个沙坑增加难度
8. **梯田挑战** - 逐级上升的球道
9. **王者之洞** - 最终挑战，综合所有元素
