# 056-Fishing 捕鱼游戏设计文档

## 📋 游戏概述

- **游戏名称**: 056-Fishing（捕鱼）
- **游戏类型**: 休闲钓鱼游戏
- **目标平台**: Windows
- **框架版本**: Zen-C 游戏框架 v1.0.0
- **开发日期**: 2026-04-06

## 🎮 游戏设计

### 核心玩法

玩家控制钓鱼竿，通过以下流程进行游戏：

1. **抛竿**：按空格键将鱼饵抛入水中
2. **等待**：鱼饵落入水中后，等待鱼靠近并咬钩
3. **反应**：看到"鱼咬钩了"提示时，快速按空格键
4. **收线**：成功抓鱼后自动收线，获得相应分数

### 钓鱼线状态机

```
Idle → Casting → Waiting → Biting → Reeling → Idle
                         ↓
                      (超时) → Idle
```

| 状态 | 描述 | 玩家操作 |
|------|------|----------|
| Idle | 等待抛竿 | 按空格抛竿 |
| Casting | 抛竿动画中 | 等待完成 |
| Waiting | 等待鱼咬钩 | 看到信号按空格 |
| Biting | 鱼咬钩了！ | 1.2秒内按空格 |
| Reeling | 收线中 | 等待完成 |

### 鱼类设计

| 鱼类型 | 颜色 (RGB) | 尺寸 | 分值 | 稀有度 |
|--------|------------|------|------|--------|
| 小鱼 | 100, 200, 100 | 12px | 10分 | 常见 |
| 中鱼 | 100, 150, 255 | 18px | 25分 | 一般 |
| 大鱼 | 150, 100, 200 | 24px | 50分 | 较少 |
| 金鱼 | 255, 215, 0 | 20px | 100分 | 稀有 |

### 鱼群行为

- 鱼在水面下方区域游动
- 碰到边界会反弹
- 随机改变游动方向
- 被抓后会重新生成

### 动画效果

1. **水面波纹**：正弦波动画，模拟水面波动
2. **抛竿动画**：从鱼竿到落点的线性插值
3. **咬钩晃动**：鱼咬钩时钩子剧烈晃动
4. **鱼游动**：持续的水平移动和边界反弹

## 📊 计分系统

- **捕获计数**：成功抓到的鱼数量
- **总分**：累计得分
- **分值加成**：金鱼给予最高100分奖励

## 🔧 技术实现

### 核心类

- **FishingGame**：继承自 Game，实现钓鱼游戏逻辑
- **Fish**：鱼的数据结构
- **LineState**：钓鱼线状态枚举

### 关键技术点

1. **状态机管理**：使用枚举管理钓鱼线状态
2. **碰撞检测**：鱼钩与鱼的位置距离检测
3. **动画插值**：抛竿和收线的平滑动画
4. **帧率控制**：60 FPS 游戏循环
