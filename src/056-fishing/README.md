# 056-Fishing 捕鱼游戏 🐟

Zen-C 游戏框架 v1.0.0 | Windows 小游戏 DIY 系列

## 🎮 游戏简介

一款经典的钓鱼小游戏，玩家通过抛竿、等待鱼咬钩、收竿的完整钓鱼流程来捕获各种鱼类。

## 📖 游戏玩法

### 操作说明

- **空格键** - 抛竿 / 抓鱼
- **鼠标左键** - 抛竿 / 抓鱼（点击水面）

### 游戏流程

1. **抛竿**：按空格键将鱼饵抛入水中
2. **等待**：观察水面，等待鱼咬钩的信号
3. **抓鱼**：看到"鱼咬钩了"提示时，快速按空格键
4. **收线**：成功抓鱼后，自动收线获得分数

### 鱼类分值

| 鱼类型 | 颜色 | 分值 |
|--------|------|------|
| 🟢 小鱼 | 绿色 | 10分 |
| 🔵 中鱼 | 蓝色 | 25分 |
| 🟣 大鱼 | 紫色 | 50分 |
| 🟡 金鱼 | 金色 | 100分 |

## 🎯 游戏特性

- **逼真的钓鱼流程**：抛竿 → 等待 → 咬钩 → 收线
- **多种鱼类**：4种不同类型的鱼，分值各异
- **时间挑战**：咬钩后需要在限定时间内反应
- **流畅动画**：水面波纹、鱼群游动等动画效果
- **计分系统**：记录捕获数量和总分
