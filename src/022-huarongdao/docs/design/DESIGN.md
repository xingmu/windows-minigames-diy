# 华容道 Klotski 游戏设计文档

## 1. 概述

### 游戏名称
- 中文名: 华容道
- 英文名: Klotski
- 项目编号: 022

### 游戏类型
经典滑块益智游戏

### 核心玩法
通过移动不同大小的方块，让最大的红色方块（代表曹操）从棋盘底部出口逃出。

### 目标用户
- 喜欢益智游戏的玩家
- 中国传统文化爱好者

## 2. 游戏规则

### 棋盘规格
- 棋盘尺寸: 4行 × 5列
- 每个单元格: 80×80 像素
- 总窗口尺寸: 440×420 像素

### 方块类型
| ID | 类型 | 尺寸 | 数量 | 颜色 |
|----|------|------|------|------|
| 0 | 曹操(大方块) | 2×2 | 1 | 红色 |
| 1-4 | 竖直武将 | 1×2 | 4 | 青色 |
| 5-6 | 横向武将 | 2×1 | 2 | 黄色 |
| 7 | 竖直武将 | 1×2 | 1 | 青色 |
| 8-9 | 士兵 | 1×1 | 2 | 浅绿 |

### 移动规则
1. 方块只能水平或垂直移动
2. 方块不能穿过其他方块
3. 方块不能超出棋盘边界

### 胜利条件
- 曹操(2×2大方块)到达位置 (2, 2)
- 该位置在棋盘底部中央，标记为绿色出口

## 3. 初始布局

```
  0   1   2   3   4
┌───┬───┬───┬───┬───┐ 0
│ V │ G G │ H │ V │ 
├───┼───┼───┼───┼───┤ 1
│ S │ G G │ S │ V │
├───┼───┼───┼───┼───┤ 2
│ V │ H H │ H │ V │
├───┴───┼───┴───┼───┤ 3
│   │ EXIT│   │   │ 
└───┴────┴───┴───┘ 
```

图例:
- G = 2×2 大方块 (曹操)
- V = 1×2 竖直方块
- H = 2×1 横向方块
- S = 1×1 士兵

## 4. 控制方案

### 鼠标拖拽
- 按下鼠标左键: 选择方块
- 拖动: 移动方块到相邻空格
- 释放: 确认移动

### 键盘快捷键
| 按键 | 功能 |
|------|------|
| R | 重新开始 |
| ESC | 退出游戏 |

## 5. 技术实现

### 数据结构

```zen-c
// 方块定义
struct Block {
    id: i32,      // 方块ID
    type: i32,    // 方块类型
    row: i32,     // 起始行
    col: i32,     // 起始列
    width: i32,   // 宽度
    height: i32,  // 高度
    color: u32,   // 颜色
}

// 棋盘状态
// 使用一维数组表示4×5棋盘
// -1 表示空格
// >=0 表示方块ID
```

### 碰撞检测
- 移动前检查目标位置所有单元格是否为空
- 边界检查防止出界

### 渲染系统
- 直接写入帧缓冲
- 60 FPS 游戏循环
- 方块带阴影和边框效果
