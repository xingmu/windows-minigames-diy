# 082-Billiards 设计文档

## 1. 项目概述

### 项目名称
- **名称**: 9-Ball Pool
- **编号**: 082
- **类型**: 2D 台球游戏
- **语言**: Zen-C
- **框架**: 003-framework

### 项目定位
与016-Billiards (斯诺克俯视图) 不同的9球普尔游戏，采用美式9球规则。

## 2. 游戏设计

### 核心玩法
- 9球制台球，使用1-9号球 + 白球
- 必须按顺序击中球 (先碰1号，再碰2号...)
- 9号球进袋获胜

### 物理系统
- 球体碰撞 (圆形碰撞检测)
- 边界反弹 (边岸碰撞)
- 摩擦衰减 (速度随时间降低)
- 落袋检测 (6个袋口)

## 3. 视觉设计

### 台球桌
- 尺寸: 900x520
- 台布颜色: 绿色 (#1C5F2D)
- 边框: 木质深色

### 球体
- 半径: 14像素
- 1-8号球: 实心彩色
- 9号球: 条纹设计
- 白球: 象牙白

### UI元素
- 力度条 (红黄绿渐变)
- 球杆 (拖拽时显示)
- 瞄准线 (虚线)

## 4. 技术架构

### 模块划分
```
src/
├── platform_api.zc     # 平台API (from 003-framework)
├── game_logic.zc       # 游戏逻辑
│   - 球数据结构
│   - 物理引擎
│   - 碰撞检测
│   - 游戏规则
└── main_window.zc       # 主窗口
    - 事件处理
    - 渲染循环
    - 画面绘制
```

### 数据结构
```zen-c
Ball {
    x, y: i32          // 位置
    vx, vy: i32        // 速度 (扩大100倍)
    radius: i32         // 半径
    color: Color        // 颜色
    number: i32         // 球号
    pocketed: bool     // 是否落袋
    active: bool        // 是否在场
    striped: bool       // 是否条纹
}

NineBallState {
    balls: [10]Ball    // 球数组
    phase: GamePhase    // 游戏阶段
    score: i32          // 分数
    power: i32          // 当前力度
}
```

## 5. 控制方案

### 鼠标操作
1. **按下**: 开始瞄准
2. **拖拽**: 调整方向 + 蓄力
3. **松开**: 执行击球

### 键盘操作
- `R`: 重新开始
- `ESC`: 退出游戏
- `Space`: 快速轻击

## 6. 状态机

```
Setup -> Dragging -> Moving -> Setup (循环)
              ↓
           Foul (犯规)
              ↓
           Setup (重置)
              ↓
            Win (9号进袋)
```

## 7. 物理参数

| 参数 | 值 |
|-----|-----|
| 球半径 | 14px |
| 袋口半径 | 22px |
| 最大力度 | 30 |
| 摩擦系数 | 1 |
| 反弹衰减 | 80% |
| 最小速度 | 1 |

## 8. 9球规则

### 开球
- 白球在左侧1/4处
- 9球三角形在右侧

### 进球
- 任意球进袋得分
- 9号球进袋直接获胜

### 犯规
- 白球落袋
- 未先碰到最小号球
- 白球空击

## 9. 渲染层次

1. 背景 (清屏)
2. 球台边框
3. 台布
4. 袋口
5. 球 (阴影 + 主体 + 高光)
6. 瞄准线 + 球杆
7. UI文字

## 10. 优化建议

- 碰撞检测使用空间分区
- 渲染时批量画圆
- 事件处理异步化
