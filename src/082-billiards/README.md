# 082-Billiards - 9-Ball Pool 🧘

经典美式9球台球游戏，使用 Zen-C + 003-framework 开发。
与016-Billiards (斯诺克) 不同，本游戏采用美式9球规则。

## 🎮 游戏特性

- **9球规则**: 经典美式9球，按顺序打入1-9号球
- **条纹9球**: 9号球带有特殊条纹设计
- **真实物理**: 球体碰撞、边界反弹、摩擦衰减
- **瞄准系统**: 拖拽瞄准，力度条显示击球力度
- **球杆显示**: 实时显示球杆位置和方向

## 🎯 游戏规则

### 9球规则
| 球号 | 颜色 | 说明 |
|-----|------|------|
| 1 | 黄色 | 第1目标球 |
| 2 | 蓝色 | 第2目标球 |
| 3 | 红色 | 第3目标球 |
| 4 | 紫色 | 第4目标球 |
| 5 | 橙色 | 第5目标球 |
| 6 | 绿色 | 第6目标球 |
| 7 | 栗色 | 第7目标球 |
| 8 | 黑色 | 第8目标球 |
| 9 | 黄条纹 | 获胜球 |

### 获胜条件
- **打入9号球** 即获得胜利!
- 打入9号球前必须先碰到序号最小的球

### 犯规情况
- 白球落袋
- 白球空击 (未碰到任何球)
- 首次碰到非最小序号球

### 控制说明
| 按键 | 功能 |
|-----|------|
| 鼠标拖拽 | 瞄准 + 蓄力 |
| 松开鼠标 | 击球 |
| R | 重新开始 |
| ESC | 退出游戏 |

## 🔧 构建说明

### Windows 构建

```bash
# 使用批处理脚本
build_windows.bat

# 或使用 Makefile
make windows
```

### Linux/macOS 构建

```bash
make
```

### 运行游戏

```bash
make run
```

## 🆚 与016-Billiards的区别

| 特性 | 016-Billiards | 082-Billiards |
|-----|---------------|---------------|
| 类型 | 斯诺克 | 9球普尔 |
| 球数 | 22球 | 10球 |
| 球号 | 0-21 | 0-9 |
| 视角 | 俯视 | 俯视 |
| 规则 | 进红球后进彩球 | 按顺序打入1-9 |
| 获胜 | 最高分获胜 | 9号球进袋获胜 |

## 🛠️ 技术栈

- **语言**: Zen-C
- **框架**: 003-framework
- **平台**: Windows (Win32 API)
- **渲染**: 帧缓冲直接写入
- **物理**: 2D圆形碰撞检测

## 📝 游戏状态

- **Setup**: 等待击球
- **Dragging**: 拖拽蓄力中
- **Moving**: 球运动中
- **Foul**: 犯规
- **Win**: 胜利

## 🎨 视觉效果

- **台布**: 深浅交替绿色纹理
- **边框**: 木质纹理效果
- **球体**: 3D高光效果
- **9号球**: 特殊条纹设计
