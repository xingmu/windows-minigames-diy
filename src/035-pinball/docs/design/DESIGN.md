# Pinball 游戏设计文档

## 概述

**项目名称**: 035-Pinball（弹球机）  
**类型**: 2D 街机物理游戏  
**目标**: 复刻经典弹球机游戏体验，模拟真实物理

## 游戏尺寸

- **宽度**: 540 像素
- **高度**: 700 像素
- **帧率**: 60 FPS (~16ms)

## 物理参数

### 球
- **半径**: 8 像素
- **重力**: 0.25 像素/帧²
- **摩擦系数**: 0.998
- **墙壁反弹系数**: 0.75
- **挡板反弹系数**: 1.2（超弹性）

### 翻转器
- **长度**: 55 像素
- **宽度**: 10 像素
- **角度速度**: 0.35 弧度/帧
- **休息角度（左）**: 0.4 弧度
- **活动角度（左）**: -0.4 弧度
- **休息角度（右）**: 3.14 - 0.4 弧度
- **活动角度（右）**: 3.14 + 0.4 弧度

### 弹簧发射器
- **最大拉动**: 40 像素
- **最大发射速度**: 18.0 像素/帧
- **发射功率范围**: 9.0 ~ 18.0 像素/帧

## 游戏元素

### 1. 球（Ball）
- 数量: 3颗
- 状态: active / inactive
- 轨迹效果: 每4帧绘制一个渐隐圆点

### 2. 翻转器（Flipper）
- 数量: 2个（左/右）
- 碰撞响应: 向上挥动时给予球额外向上的速度

### 3. 挡板（Bumper）
- 数量: 5个
- 排列: 三角形
- 碰撞: 超弹性反弹（1.2倍）
- 击中效果: 黄色高亮 10 帧

### 4. 目标（Target）
- 数量: 6个
- 排列: 水平一排
- 击中后: 变灰，20帧后重置
- 全部清除: 倍数+1

### 5. 弹弓（Slingshot）
- 数量: 2个
- 形状: 三角形
- 反弹系数: 1.1

### 6. 旋转器（Spinner）
- 位置: 台面中部
- 功能: 球经过时旋转并得分
- 每5次旋转计一次分

## 得分系统

| 元素 | 单次得分 |
|------|----------|
| 挡板（Bumper） | 100 × 倍数 |
| 目标（Target） | 500 × 倍数 |
| 弹弓（Slingshot） | 50 × 倍数 |
| 旋转器（Spinner） | 25 × 倍数 |

**倍数**: 清空所有目标后 +1，最高 5 倍

## 操作映射

| 键盘事件 | 功能 |
|----------|------|
| KeyDown[Space] | 开始拉弹簧 |
| KeyDown[Up] | 增加拉力 |
| KeyDown[Down] | 减少拉力 |
| KeyUp[Space] | 释放弹簧发射 |
| KeyDown[Left] | 左翻转器向上 |
| KeyUp[Left] | 左翻转器向下 |
| KeyDown[Right] | 右翻转器向上 |
| KeyUp[Right] | 右翻转器向下 |
| KeyDown[R] | 重新开始 |
| KeyDown[Escape] | 退出游戏 |

## 碰撞检测

### 算法
1. **球 vs 圆**: 距离检测 + 法线反弹
2. **球 vs 线段**: 点到线段最近点 + 分离
3. **球 vs 翻转器**: 端点追踪 + 速度加成
4. **球 vs 矩形**: 最近点检测 + 轴对齐反弹

### 碰撞响应
- 分离: 将球推出重叠区域
- 反弹: 使用法线反射，速度乘以弹性系数
- 翻转器特殊: 向上挥动时额外添加 -6 像素/帧的垂直速度
