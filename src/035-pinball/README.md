# 035-Pinball（弹球机） 🏓

Zen-C 语言开发的经典弹球机游戏，模拟真实弹球机物理。

## 🎮 游戏特性

- **真实物理模拟** - 重力、摩擦力、碰撞反弹
- **翻转器控制** - 左右翻转器，弹球机核心玩法
- **弹簧发射器** - 拉弹簧蓄力发射弹珠
- **挡板（Bumper）** - 圆形挡板，被击中会弹开并得分
- **下落目标（Target）** - 一排目标，被击中全部清除后倍数提升
- **弹弓（Slingshot）** - 三角形弹射装置
- **旋转器（Spinner）** - 经过可旋转得分
- **多球系统** - 3颗球，掉球不立即结束

## 🕹️ 操作说明

| 按键 | 功能 |
|------|------|
| `空格` | 按住拉弹簧，松开发射 |
| `↑` | 增加发射力度 |
| `↓` | 减少发射力度 |
| `←` | 左翻转器向上 |
| `→` | 右翻转器向上 |
| `R` | 重新开始 |

## 📐 游戏规则

- **挡板得分**: 每击中一个挡板 +100分 × 倍数
- **目标得分**: 每个目标 +500分 × 倍数
- **弹弓得分**: 每次弹射 +50分 × 倍数
- **倍数系统**: 清空所有目标后倍数+1，最高5倍
- **球数**: 每局3颗球，掉球不立即结束
- **游戏结束**: 3颗球全部丢失
