# 061-Racing - 俯视角赛车游戏

Zen-C + 003-framework 实现

## 游戏介绍

经典俯视角赛车游戏，玩家驾驶赛车在赛道上竞速。

## 操作说明

- **W / ↑**: 加速
- **S / ↓**: 刹车/倒车
- **A / ←**: 左转
- **D / →**: 右转
- **空格键**: 暂停
- **ESC**: 退出

## 游戏特性

- 俯视角赛车控制
- 计时竞速模式
- 圈数统计
- 速度仪表盘
- 赛道边界碰撞检测

## 编译运行

### Windows

```bash
build_windows.bat
```

### Linux/macOS

```bash
make
```

## 技术实现

- 框架: Zen-C 003-framework
- 语言: Zen-C
- 平台: Windows (Linux/macOS via Makefile)
