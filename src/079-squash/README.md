# 079-Squash 壁球游戏

Zen-C 游戏框架下的壁球游戏，玩家与电脑对战。

## 游戏说明

壁球是一项在封闭场地内进行的球拍运动，球通过墙壁反弹。本游戏模拟了这一经典运动。

### 操作说明

- **鼠标移动**: 控制球拍左右前后移动
- **鼠标点击**: 发球
- **R键**: 游戏结束后重新开始

### 游戏规则

1. 球必须首先击中前墙
2. 球在地板上弹跳超过2次，对方得分
3. 先得到9分者获胜
4. 玩家有3条生命值

### 游戏特性

- 真实的物理模拟（重力、反弹、摩擦）
- 智能AI对手
- 渐进式难度（球速逐渐增加）
- 3D透视视角
