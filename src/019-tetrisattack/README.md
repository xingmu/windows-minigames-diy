# TetrisAttack 俄罗斯方块消消乐

Zen-C 语言开发的经典俄罗斯方块消消乐游戏。

## 游戏规则

- **目标**: 在限定时间内，尽可能获得更高的分数
- **消除规则**: 当 **3个或更多** 相同颜色的方块水平或垂直相连时，自动消除
- **连锁反应**: 消除后方块在重力作用下下落，可能引发连锁消除（连击）
- **计分**: 
  - 每消除一个方块 +10 分
  - 连击额外奖励 +5 分/每连击

## 操作说明

| 操作 | 功能 |
|------|------|
| ← → | 移动下落方块 / 移动光标 |
| ↑ | 交换两个方块的颜色 |
| ↓ | 快速下落 |
| 鼠标点击 | 移动光标到指定列 |
| R | 重新开始游戏 |
| ESC | 退出游戏 |

## 游戏特性

- 12×18 方块网格
- 6 种不同颜色的方块
- 连锁消除（连击）系统
- 实时计时器
- 分数统计
- 光标闪烁指示

## 开发

游戏使用帧缓冲 (Framebuffer) 进行渲染，直接写入显存实现高效绘制。

### 核心算法

- **消除检测**: 遍历所有行和列，查找3个或以上连续相同颜色
- **重力系统**: 从下往上检测空位，让方块下落填充
- **连锁反应**: 消除 → 重力 → 再次检测消除 → 重复直到稳定
