# Gobang 五子棋

Zen-C 语言开发的经典五子棋游戏，玩家 VS 电脑 AI。

## 游戏规则

- **棋盘**: 15×15 格
- **玩法**: 黑棋先行，玩家执黑；白棋为电脑 AI
- **胜利条件**: 横、竖、斜任意方向连成 **五子连珠**
- **落子方式**: 鼠标点击棋盘交叉点

## 操作说明

| 操作 | 功能 |
|------|------|
| 鼠标点击 | 在交叉点落子 |
| R | 重新开始游戏 |
| ESC | 退出游戏 |

## 游戏特性

- 15×15 标准棋盘
- 玩家执黑棋先手
- AI 电脑对手（白棋）
- 威胁评估算法：自动识别冲四、活三、眠三等棋型
- 最后落子高亮显示
- 获胜连线标注
- 实时回合提示

## AI 算法

采用威胁评估算法对每个空位进行评分：

| 棋型 | 分数 | 说明 |
|------|------|------|
| 五连 | 100000 | 直接获胜 |
| 活四/冲四 | 10000 | 即将成五 |
| 活三 | 800 | 威胁极大 |
| 冲三 | 200 | 有一定威胁 |
| 活二 | 100 | 发展潜力 |
| 眠二 | 50 | 较弱棋型 |

AI 同时评估进攻（白棋）和防守（黑棋）的最优着法，优先处理高威胁局面。

## 开发

游戏使用帧缓冲 (Framebuffer) 直接写入显存进行渲染。

### 核心模块

- **game_logic.zc**: 棋盘状态管理、落子逻辑、获胜检测、AI 评估
- **main_window.zc**: 窗口管理、事件处理、棋盘渲染

### 获胜检测

检查四个方向（横、竖、左斜、右斜），每方向统计连续己方棋子数，达到 5 即获胜。
