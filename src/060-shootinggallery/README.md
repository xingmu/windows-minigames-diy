# 060-ShootingGallery 射击训练游戏

> Zen-C + 003-framework 开发的移动靶射击训练游戏

## 游戏简介

**射击训练游戏 (Shooting Gallery)** 是一款60秒限时射击挑战游戏。玩家需要在规定时间内尽可能多地击中移动的靶子，获取最高分数。

## 游戏特色

- **三种靶子类型**:
  - 🟥 **站立靶** (红色) - 10分，固定不动
  - 🟦 **移动靶** (蓝色) - 20分，在区域内移动
  - 🟨 **奖励靶** (金色) - 50分，稀有目标

- **60秒限时挑战**: 考验玩家的射击速度和准确度
- **实时数据统计**: 显示分数、剩余时间和命中率
- **流畅的帧率**: 60FPS 稳定运行

## 操作说明

| 操作 | 按键/鼠标 |
|------|----------|
| 瞄准 | 鼠标移动 |
| 射击 | 左键点击 |
| 退出 | ESC |

## 系统要求

- Windows 操作系统
- Zen-C 运行时环境

## 快速开始

### 编译运行 (Windows)

```bash
# 使用构建脚本
build_windows.bat

# 或使用 Makefile (需要 Make 工具)
make
./build/shootinggallery
```
