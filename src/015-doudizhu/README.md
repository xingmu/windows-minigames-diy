# 015-Doudizhu 斗地主

Zen-C 语言开发的经典斗地主游戏，基于 003-framework 框架。

## 🎮 游戏介绍

经典斗地主游戏，玩家 vs 电脑AI，支持完整斗地主规则。

## 📋 游戏规则

### 发牌
- 54张扑克牌（52张点数牌 + 大王 + 小王）
- 三名玩家各发17张牌，3张底牌
- 底牌归地主所有

### 叫地主
- 玩家轮流叫分（1-3分）或放弃
- 叫分最高者成为地主，获得底牌
- 无人叫分则重新发牌

### 出牌
- 地主先出牌，之后按逆时针顺序出牌
- 必须出比上家大的牌，或选择"过"
- 连续两家"过"后，可以出任意合法牌型

### 牌型大小
- **火箭（王炸）**：大王 + 小王，最大
- **炸弹**：四张同点数
- 其他牌型：必须类型相同且张数相同才能比较
- 单张、对子、三张：点数大者胜
- 顺子、连对、飞机：5张/3对/2个3以上，连续且张数相同
- 三带一、三带二、四带二：按三张/四张的点数比较

## 🕹️ 操作说明

### 鼠标
- 点击卡牌选中/取消选中
- 点击"叫分"按钮进行叫地主

### 键盘
- `空格`：出牌（先选中牌）
- `Q`：过（跟牌时）
- `R`：重新开始（游戏结束后）
- `ESC`：退出游戏
