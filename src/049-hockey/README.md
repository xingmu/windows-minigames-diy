# 049-Hockey - 冰球守门员游戏

Zen-C 冰球守门员游戏，扮演守门员用鼠标控制手套和球杆扑救冰球。

## 🎮 游戏介绍

一款冰球守门员模拟游戏，玩家控制守门员在球门前扑救飞来的冰球，体验紧张的守门员角色。

### 游戏特性

- **守门员视角** - 玩家扮演守门员角色
- **双手装备** - 左手手套(挡板) + 右手球杆
- **鼠标控制** - 移动鼠标控制守门员位置
- **难度递增** - 随着分数增加，冰球速度和数量增加
- **生命系统** - 3条生命，失去所有生命游戏结束

## 🕹️ 操作说明

| 操作 | 功能 |
|------|------|
| 鼠标移动 | 控制守门员左右移动 |
| 鼠标左键 | 确认位置 |
| R键 | 重新开始游戏 |

## 📐 游戏规则

- 冰球从场地顶部飞向球门
- 用守门员的身体、手套或球杆扑救冰球
- 成功扑救得1分
- 冰球越过球门线失去1条生命
- 获得30分获胜
- 每得10分升级，难度提升

## 🛠️ 构建说明

### Zen-C 编译器

需要安装 Zen-C 编译器 `zc`。

### Windows 构建

```bash

```

### Linux/macOS 构建

```bash

./build/hockey_game
```

## 🎯 游戏界面

- **顶部** - 冰球发射区域
- **中间** - 冰球场地区域
- **底部** - 球门和守门员
- **左上** - 分数和生命显示
- **右上** - 当前关卡和目标

## 🏒 游戏元素

| 元素 | 描述 |
|------|------|
| 守门员 | 主要扑救区域，需要覆盖球门宽度 |
| 手套(左手) | 辅助扑救，可以伸出去挡球 |
| 球杆(右手) | 辅助扑救，blade可以挡球 |
| 冰球 | 从顶部飞向球门 |
| 球门 | 红色门框和白色球门线 |
| 冰面 | 淡蓝色冰场背景 |
