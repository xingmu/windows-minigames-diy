# BreakoutPlus - 打砖块增强版 🧱

Zen-C 多关卡打砖块游戏，包含特殊道具系统。

## 🎮 游戏特色

### 多关卡系统
- **5个精心设计的关卡**
- 每关砖块布局、密度、球速递增
- 关卡进度显示在屏幕底部

### 特殊道具系统
| 道具 | 字符 | 效果 |
|------|------|------|
| 扩展挡板 | E | 挡板宽度增加50%，持续10秒 |
| 缩小挡板 | S | 挡板宽度缩小40%，持续10秒 |
| 多球 | M | 分裂出最多3个额外球 |
| 减速球 | L | 所有球速度降低 |
| 加速球 | F | 所有球速度增加 |
| 额外生命 | + | 获得1条额外生命 |
| 粘性挡板 | K | 球会粘在挡板上，再按空格发射 |
| 激光 | X | 挡板发射激光，可远程消灭砖块 |

### 视觉效果
- 粒子爆炸效果
- 动态砖块高光/阴影
- 道具发光效果
- 背景网格动画

## 🎯 游戏规则

- 使用 **← →** 方向键移动挡板
- 按 **空格键** 开始游戏/发射球
- 按 **P** 暂停/继续
- 按 **ESC** 退出游戏
- 按 **R** 在游戏结束后重新开始

### 计分规则
- 消灭砖块获得分数（上层砖块分值更高）
- 使用激光消灭砖块得5分
- 完成所有5关即为大师

## 🔧 构建说明

### Windows 构建

```bash
build_windows.bat
```

或使用 Zen-C 编译器：

```bash
zc src/main_window.zc -o build/breakoutplus.exe
```

### Linux/macOS 构建

```bash
make
```
