# Match3 消消乐游戏设计文档

## 1. 游戏概述

**游戏名称**: Match3 消消乐  
**游戏类型**: 休闲益智游戏  
**目标用户**: 全年龄层玩家  
**核心玩法**: 发射彩球，三个或以上相同颜色连成一线即可消除

## 2. 游戏规则

### 2.1 基本规则

- 游戏区域为一个 8x10 的网格
- 玩家从屏幕底部向上发射彩球
- 彩球落在网格中，如果有三个或以上相同颜色的球连成水平或垂直线，则消除
- 消除后，上方的球在重力作用下下落填补空白

### 2.2 计分规则

- 消除一个方块: 10 分
- 连击 bonus: 每个额外连击 +5 分
- 示例: 3连击 = 30分, 4连击 = 30+5 = 35分, 5连击 = 30+10 = 40分

### 2.3 时间限制

- 游戏时间: 2 分钟 (120 秒)
- 时间耗尽后游戏结束

## 3. 技术设计

### 3.1 技术栈

- **编程语言**: Zen-C
- **图形 API**: Windows GDI
- **游戏框架**: 003-framework

### 3.2 数据结构

```zen-c
struct Ball {
    x: f32,           // 球的位置
    y: f32,
    vx: f32,          // 速度
    vy: f32,
    active: bool,    // 是否在移动
    color_index: i32, // 颜色索引 (0-5)
}

struct GridCell {
    color_index: i32, // -1 表示空
}

struct Match3State {
    grid: [10][8]GridCell, // 游戏网格
    ball: Ball,            // 当前球
    next_color: i32,       // 下一个球颜色
    score: i32,            // 分数
    time_remaining: i32,  // 剩余时间
    status: GameStatus,   // 游戏状态
    combo: i32,           // 连击数
}
```

### 3.3 游戏状态

```zen-c
enum GameStatus {
    Ready,      // 等待开始
    Playing,    // 游戏中
    Paused,     // 暂停
    GameOver,   // 游戏结束
}
```

## 4. 核心算法

### 4.1 发射机制

1. 玩家点击屏幕
2. 计算从发射器到点击位置的单位向量
3. 设置球的速度向量
4. 激活球，开始移动

### 4.2 碰撞检测

- **墙壁碰撞**: 左右墙壁反弹
- **顶部碰撞**: 球嵌入网格

### 4.3 匹配检测

1. 遍历所有行，检查水平方向连续相同颜色
2. 遍历所有列，检查垂直方向连续相同颜色
3. 标记所有匹配的格子
4. 清除匹配的格子

### 4.4 重力掉落

1. 对于每一列，从底部向上收集非空块
2. 从底部开始填回网格
3. 顶部清空

## 5. 用户界面

### 5.1 布局

```
+----------------------------------+
|  分数: 0           时间: 2:00   |
+----------------------------------+
|                                  |
|   +------------------------+    |
|   |                        |    |
|   |      8x10 游戏网格      |    |
|   |                        |    |
|   |                        |    |
|   +------------------------+    |
|                                  |
|           (O) NEXT              |
|           [  ●  ]                |
|                                  |
+----------------------------------+
|  控制: 点击发射 | 空格开始 | ESC |
+----------------------------------+
```

### 5.2 控制

- **鼠标点击**: 发射彩球（只能向上发射）
- **空格键**: 开始/重新开始游戏
- **ESC**: 退出游戏
