# 070-Golf 高尔夫游戏

Zen-C 编写的高尔夫游戏，支持鼠标力度方向击球。

## 游戏特性

- 🎯 鼠标拖拽瞄准 - 点击并拖拽控制击球方向和力度
- 🏌️ 4个不同难度的球洞
- 🏳️ 沙坑障碍 - 球进入沙坑会减速
- 💧 水障碍 - 球进入水障碍会被重置
- 📊 杆数统计 - 记录每洞杆数和总杆数
- 🚩 旗杆标记 - 清晰显示球洞位置

## 操作说明

1. **瞄准**: 在球上点击并按住鼠标
2. **蓄力**: 拖拽鼠标远离球，蓄力条会显示力度
3. **击球**: 松开鼠标，球会向拖拽的反方向飞出
4. **进洞**: 将球推入洞中完成当前洞

## 控制

- **鼠标拖拽**: 瞄准和蓄力
- **ESC键**: 退出游戏
- **点击**: 进洞后进入下一洞/重新开始

## 构建

```bash
# 编译游戏
make all

# 编译并运行
make run

# 编译Windows版本
make windows

# 清理
make clean
```

## 技术细节

- 使用 Zen-C 语言开发
- 基于 003-framework 游戏框架
- 帧缓冲直接渲染
- 简单的2D物理引擎

## 游戏规则

- 每洞有标准杆数(Par)
- 记录击球次数
- 尽量用最少杆数完成所有球洞
