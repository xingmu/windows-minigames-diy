# 063-Boat 赛艇游戏设计文档

## 概述

赛艇大赛是一款2D俯视角水上竞速游戏，玩家操控赛艇在水上赛道中与AI对手竞速，率先到达终点即获胜。

## 游戏设计

### 画面布局

```
+--------------------------------------------------+
|  赛艇大赛                        时间: 00:00.00  |
|  距离终点: 500米                                   |
+--------------------------------------------------+
|     |  1   |  2   |  3   |  4   |                  |  <- 4车道
|     |      |      |      |      |                  |
|     |  [障碍物]   |      |      |                  |  <- 障碍物
|     |      |      | [障碍物]   |                  |
|     |      |      |      |      |                  |
|     |      |      |      |      |                  |
|     |  [终点线 - 黑白格子]  |                  |  <- 终点线
|     |      |      |      |      |                  |
|     |  [玩家赛艇] |      |      |                  |  <- 玩家
|     |      | [AI1] |      |      |                  |  <- AI对手
|     |      |      | [AI2] | [AI3] |                  |
+--------------------------------------------------+
```

### 游戏常量

| 常量 | 值 | 说明 |
|------|-----|------|
| WINDOW_WIDTH | 800 | 窗口宽度 |
| WINDOW_HEIGHT | 600 | 窗口高度 |
| LANE_COUNT | 4 | 车道数量 |
| LANE_WIDTH | 80 | 单车道宽度 |
| TRACK_LEFT | 120 | 赛道左边界 |
| TRACK_RIGHT | 440 | 赛道右边界 |
| WATER_TOP | 50 | 水面顶部 |
| WATER_BOTTOM | 550 | 水面底部 |
| BOAT_WIDTH | 50 | 赛艇宽度 |
| BOAT_HEIGHT | 20 | 赛艇高度 |
| ROWING_SPEED | 3 | 划船前进速度 |
| OBSTACLE_COUNT | 6 | 障碍物数量 |
| FINISH_LINE_Y | 80 | 终点线Y坐标 |

### 玩家控制

- **← → / A D**: 左右移动
- **空格键**: 划船前进（带冷却时间300ms）
- **R**: 游戏结束后重新开始
- **鼠标点击**: 开始游戏

### 游戏机制

#### 物理系统

- 船只移动带有惯性
- 划船产生前进力
- 左右移动有阻力衰减
- 碰撞障碍物速度大幅降低

#### AI系统

- 3艘AI赛艇，不同速度和节奏
- AI划船周期: 400ms + 100ms*i
- AI偶尔左右偏移模拟真实操作
- AI互不碰撞

#### 障碍物

- 6个木桩/岩石障碍
- 缓慢向下移动
- 碰撞后速度降低50%
- 玩家会被推动向下

### 渲染层次

1. 天空背景（渐变）
2. 水面（带颜色渐变）
3. 波浪效果
4. 赛道边框
5. 车道分隔线
6. 障碍物
7. AI赛艇
8. 玩家赛艇（带旗帜标记）
9. 终点线
10. HUD信息
11. 消息覆盖层

### 胜负判定

- **胜利**: 玩家率先到达终点线（y <= FINISH_LINE_Y + 5）
- **结束**: 所有船只完成比赛

### 计时系统

- 格式: MM:SS.mm（分:秒.毫秒）
- 玩家开始时间: 按空格键时刻
- 赛艇用时: finishTime - startTime

## 技术实现

### 数据结构

```zen-c
struct Boat {
    x, y: float        // 位置
    vx, vy: float       // 速度
    lane: int           // 所在车道
    rowing: bool        // 是否正在划船
    rowingTimer: u32    // 划船计时
    rowingFrame: int    // 划船动画帧
    color: u32          // 船体颜色
    finished: bool      // 是否完成
    finishTime: u32     // 完成时间
    startTime: u32      // 开始时间
}

struct Obstacle {
    x, y: float         // 位置
    width, height: int  // 尺寸
    lane: int           // 所在车道
    active: bool        // 是否激活
    vy: float           // 下移速度
    color: u32          // 颜色
}

struct WavePoint {
    x, y: float         // 位置
    phase: float        // 相位
    amp: float          // 振幅
}
```

### 动画系统

- 波浪: 32个点，正弦波动
- 划桨: 动画帧切换，划桨角度变化
- 水面: 颜色随位置和时间变化
