# LinkLink 连连看游戏

Zen-C 语言开发的经典连连看游戏。

## 游戏规则

- **目标**: 在限定时间内，消除所有成对的相同方块
- **连接规则**: 两个相同方块必须能在最多 **2 个拐角** 内连通
- **计分**: 
  - 每消除一对 +100 分基础分
  - 时间奖励（剩余时间越多分数越高）
  - 路径奖励（拐角越少分数越高）

## 操作说明

| 操作 | 功能 |
|------|------|
| 鼠标点击 | 选择/消除方块 |
| R | 重新开始游戏 |
| ESC | 退出游戏 |

## 游戏特性

- 8×8 方块网格
- 18 种不同图案（配对出现）
- 彩色圆角矩形方块
- 绿色连接线显示消除路径
- 实时计时器
- 分数统计

## 开发

游戏使用帧缓冲 (Framebuffer) 进行渲染，直接写入显存实现高效绘制。

### 核心算法

- **路径查找**: BFS/DFS 算法，检测两点间是否可在最多2个拐角内连通
- **连接检测**: 
  1. 直接连通（直线无阻碍）
  2. 一个拐角（需要1个转折点）
  3. 两个拐角（需要2个转折点）
