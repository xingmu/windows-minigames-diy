# 093-TowerOfHanoi - 汉诺塔游戏 🧘

Zen-C 语言实现的经典汉诺塔游戏，基于 003-framework 框架。

## 🎮 游戏简介

汉诺塔是一个经典的递归算法问题。游戏目标是将所有盘子从杆 A 移动到杆 C，可以利用杆 B 作为辅助，但必须遵守以下规则：

1. 每次只能移动一个盘子
2. 盘子只能从杆子顶端取出
3. 较大的盘子不能放在较小的盘子上面

## 🎯 游戏目标

用最少的移动次数将所有盘子从杆 A 移动到杆 C。

**最少移动次数公式**: `2^n - 1`（n 为盘子数量）

| 盘子数 | 最少移动次数 |
|--------|-------------|
| 3      | 7           |
| 4      | 15          |
| 5      | 31          |
| 6      | 63          |
| 7      | 127         |
| 8      | 255         |

## ⌨️ 操作说明

| 按键 | 功能 |
|------|------|
| `1` / `2` / `3` | 直接选择杆子 A / B / C |
| `A` / `D` | 选择左/右边的杆子 |
| `↑` / `W` / `空格` | 移动选中的盘子 |
| `R` | 重新开始游戏 |
| `N` | 新游戏（增加盘子数） |
| `M` | 新游戏（减少盘子数） |
| `P` | 暂停/继续 |
| `ESC` | 退出游戏 |
