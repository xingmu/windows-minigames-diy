# 055-Wrestling（职业摔跤）👊

Zen-C + 003-framework 实现的职业摔跤风格对战游戏。

## 🎮 游戏特色

- **职业摔跤招式系统** - 抓举、过肩摔、身体砸击、压制
- **能量管理系统** - 消耗能量释放技能，能量不足时自动恢复
- **双人对战** - 玩家 vs CPU
- **2局2胜制** - 真正的摔跤比赛规则
- **绳索弹跳** - 撞到绳索会弹回
- **压制判定** - 压制倒地对手3秒即可获胜

## 🎮 操作说明

### 玩家1 (左侧)

| 按键 | 动作 |
|------|------|
| ← → | 移动 |
| ↑ | 跳跃 |
| A | 抓举/攻击 (靠近敌人时) |
| B | 特殊技 |
| C | 闪避 |
| A+C | 压制 (在倒地的敌人上) |
| B+C | 嘲讽 (恢复能量) |

### 组合键

| 组合 | 招式 | 伤害 | 能量消耗 |
|------|------|------|----------|
| 靠近+A | 抓举 | 8 | 10 |
| ↑+A | 身体砸击 | 15 | 20 |
| ↓+A | 抛摔 | 10 | 15 |
| ↑+↓+B | 过肩摔 | 30 | 35 |
| ←+→+B | 爱尔兰鞭 | 20 | 25 |
| B+C | 终结技 | 40 | 50 |
| 靠近+A+C | 压制 | - | 20 |

## 🕹️ 游戏规则

1. 双方各有 **100点血量** 和 **100点能量**
2. **抓举**: 靠近敌人按A可以抓住敌人进行抓举
3. **特殊技**: 消耗能量释放强力技能
4. **闪避**: C键向后闪避，消耗少量能量
5. **压制**: 在倒地的敌人身上按A+C，压制3秒获胜
6. **嘲讽**: 同时按B+C可以嘲讽恢复能量
7. **能量**: 不使用技能时自动恢复
8. 先赢 **2局** 者获得比赛胜利
