# Snowboard 游戏设计文档

## 概述

Snowboard 是一款俯视角单板滑雪游戏，玩家在雪道上躲避障碍物，尽可能滑得更远。

## 游戏机制

### 核心玩法

- 玩家从雪道顶端向下滑行
- 通过左右移动躲避障碍物
- 速度随时间逐渐加快
- 障碍物包括树木、石头、旗帜
- 碰撞障碍物则游戏结束

### 计分系统

- 每成功通过一个障碍物 +1 分
- 分数实时显示在屏幕上
- 游戏结束时显示最终得分

### 难度曲线

```
时间(秒)  速度(px/s)
0         280
30        330
60        380
90        430
...       ...
180+      600 (上限)
```

## 视觉效果

### 场景层次

1. 天空渐变背景
2. 雪道区域
3. 障碍物
4. 飘雪粒子
5. 玩家
6. HUD (分数/速度)

### 颜色方案

| 元素 | 颜色 |
|------|------|
| 天空 | RGB(135,175,220) → RGB(200,220,240) |
| 雪道 | RGB(210,230,245) |
| 树木 | RGB(30,120,40) / RGB(20,100,30) |
| 石头 | RGB(100,100,110) |
| 滑雪者 | RGB(220,40,40) / RGB(40,80,180) |
| 旗帜 | 红色/白色条纹 |

### 动画效果

- 滑雪者左右倾斜
- 飘雪粒子
- 雪道划痕效果

## 技术实现

### 架构

- 使用 003-framework 框架
- Game 类继承模式
- 事件驱动消息循环
- 帧缓冲直接写像素

### 渲染流程

1. 清屏
2. 绘制背景（天空渐变 + 雪道）
3. 绘制粒子
4. 绘制障碍物
5. 绘制玩家
6. 绘制 HUD
7. presentFrame 提交

### 碰撞检测

使用 AABB (Axis-Aligned Bounding Box) 算法：

```c
if (px - pw/2 < ox + ow &&
    px + pw/2 > ox &&
    py < oy + oh &&
    py + ph > oy) {
    // 碰撞
}
```
