# 033-FishGame - 捕鱼达人

Zen-C 语言开发的捕鱼达人风格游戏，使用 003-framework 框架。

## 🎮 游戏特色

- **鼠标瞄准射击** - 用鼠标控制炮台方向，左键发射炮弹
- **多种鱼类** - 不同大小、速度的鱼群，捕获获得不同分数
- **技能道具** - 散弹、加农炮等特殊武器
- **无限模式** - 在鱼潮中尽可能获得高分

## 🕹️ 操作说明

| 操作 | 功能 |
|------|------|
| 鼠标移动 | 调整炮台射击方向 |
| 左键点击 | 发射炮弹 |
| ESC | 退出游戏 |

## 🐟 鱼种分类

| 鱼种 | 颜色 | 分数 | 速度 |
|------|------|------|------|
| 小鱼 | 绿色 | 10 | 快 |
| 中鱼 | 蓝色 | 25 | 中 |
| 大鱼 | 金色 | 50 | 慢 |
| 鲨鱼 | 红色 | 100 | 中 |
| 金鱼 | 彩虹 | 200 | 快 |

## 📁 项目结构

```
033-fish/
├── src/
│   ├── platform_api.zc         # 平台接口 (复制自003-framework)
│   ├── platform_api_windows.zc
│   ├── game_logic.zc          # 游戏逻辑：鱼、炮弹、碰撞检测
│   ├── main_window.zc         # 主程序入口
│   └── fish_render.zc         # 鱼的绘制函数
├── build/                     # 构建输出
├── Makefile
├── build_windows.bat
└── README.md
```

## 🔧 构建方法

### Windows

```bash
build_windows.bat
```

### Linux/Mac (Makefile)

```bash
make
```
