# Cricket 板球游戏设计文档

## 1. 概述

### 游戏名称
Cricket (板球)

### 游戏类型
体育运动游戏 - 击球类

### 游戏简介
玩家作为击球手，与电脑投球手进行对战，体验板球运动的乐趣。

## 2. 游戏规则

### 基本规则
- 电脑投球手向玩家投球
- 玩家鼠标点击挥杆击球
- 根据击球质量和时机获得1-6分
- 三次未击中或未挥杆则出局
- 2轮(overs)，每轮6个球

### 投球类型
1. **Normal** - 普通球：标准速度和轨迹
2. **Fast** - 快速球：速度快，弹跳低
3. **Spin** - 旋转球：横向偏移大
4. **Bouncer** - 反弹高球：垂直弹跳高

### 击球结果
- **Dot Ball** - 0分，未击中或无效击球
- **Single** - 1分
- **Double** - 2分
- **Triple** - 3分
- **Four** - 4分，边界球
- **Six** - 6分，六分球
- **Out** - 出局

## 3. 界面设计

### 布局
- **上方区域** - 投球手位置
- **中间区域** - 球场
- **下方区域** - 击球手和门柱
- **左上角** - 分数板

### 球场元素
- 绿色草地背景
- 土棕色投手道
- 三根门柱 + 两根横木
- 白色边界线

## 4. 操作控制

### 鼠标控制
- **移动鼠标** - 调整击球位置
- **鼠标左键点击** - 挥杆击球
- **R键** - 重新开始游戏

## 5. 游戏流程

### 阶段
1. **Ready** - 等待投球
2. **Bowling** - 投球动画
3. **BallBouncing** - 球弹跳
4. **BallFlying** - 球飞行
5. **Batting** - 击球判定
6. **Result** - 显示结果
7. **GameOver** - 游戏结束

## 6. 技术实现

### 框架
- Zen-C 游戏框架 v1.0.0
- 003-framework 基础类

### 核心类
- `CricketGame` - 主游戏类
- `Game` - 框架基类
- `Canvas` - 绘图接口

### 关键算法
- 球弹跳物理模拟
- 击球碰撞检测
- 投球类型随机选择
