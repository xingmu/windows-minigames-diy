# 086-Soccer - Zen-C 足球游戏

俯视角足球对战游戏，使用 Zen-C 语言和 003-framework 框架开发。

## 🎮 游戏简介

一款俯视角的2D足球游戏，玩家控制蓝色球员与AI红色球员进行足球对战。

## 🕹️ 操作说明

| 按键 | 功能 |
|------|------|
| W | 向上移动 |
| A | 向左移动 |
| S | 向下移动 |
| D | 向右移动 |
| 空格 | 踢球 |
| P | 暂停/继续 |
| R | 重新开始 |
| ESC | 退出游戏 |

## 🎯 游戏目标

- 控制你的球员带球进入对方球门得分
- 玩家（蓝色）从底部向上进攻
- AI（红色）从顶部向下进攻
- 比赛时间90秒，进球多者获胜

## 🔧 构建说明

本项目使用 Zen-C 编译器构建，需要先安装 Zen-C 工具链。

### Windows 构建

```bash
cd 086-soccer
zc main_window.zc -o soccer.exe
```

### 运行

```bash
./soccer.exe
```

## 🎨 游戏特性

- **流畅的控制**：WASD 方向键控制球员移动
- **真实的物理**：球有摩擦力和碰撞效果
- **智能AI**：AI球员会追逐球并尝试射门
- **进球判定**：精确的球门碰撞检测
- **比赛系统**：包含进球庆祝和比赛结束画面

## 📝 开发说明

- 框架：003-framework
