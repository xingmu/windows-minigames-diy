# 080-Shuffleboard 沙狐球 🧩

## 游戏简介

Zen-C 开发的沙狐球 (Shuffleboard) 游戏。玩家通过鼠标拖拽蓄力，发射圆盘滑向计分区，根据停靠位置获得相应分数。

## 游戏规则

- **目标**: 先达到 15 分的玩家获胜
- **操作**: 鼠标拖拽蓄力，松手发射圆盘
- **计分**: 靶心 10 分，逐圈递减到 5 分
- **回合**: 双方各投 4 个圆盘后计分

## 操作指南

| 操作 | 说明 |
|------|------|
| 鼠标拖拽 | 蓄力（拖拽距离决定力度） |
| 松开鼠标 | 发射圆盘 |
| ESC | 退出游戏 |

## 计分规则

```
靶心圆形区: 10 分
内圈: 8 分
第二圈: 7 分
第三圈: 6 分
外圈: 5 分
```

## 技术特性

- **语言**: Zen-C 现代系统编程
- **框架**: 003-framework 游戏框架
- **渲染**: 帧缓冲直接写入，高效绘制
- **物理**: 摩擦力衰减模拟真实滑动
