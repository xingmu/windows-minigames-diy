# 080-Shuffleboard 沙狐球设计文档

## 1. 游戏概述

沙狐球 (Shuffleboard) 是一种经典的桌上/地面滑板游戏，玩家将圆盘滑向计分区，根据停靠位置得分。

## 2. 游戏规则

### 2.1 基本规则
- 两名玩家轮流发射圆盘
- 每位玩家每回合有 4 个圆盘
- 双方投完后结算本回合分数
- 先达到 15 分的玩家获胜

### 2.2 计分规则
- 靶心 (金色): 10 分
- 内圈 (银色): 8 分
- 第二圈 (铜色): 7 分
- 第三圈 (蓝紫色): 6 分
- 外圈 (浅绿色): 5 分

### 2.3 物理规则
- 圆盘发射后沿直线滑动
- 摩擦力使速度逐渐衰减
- 圆盘可撞击其他圆盘改变位置
- 出界/压线按实际位置计分

## 3. 技术实现

### 3.1 坐标系
```
球道宽度: 600 像素
球道高度: 700 像素
靶心位置: (COURT_X + COURT_WIDTH/2, COURT_Y + 60)
发射线位置: SHOOTER_Y = COURT_Y + COURT_HEIGHT - 60
```

### 3.2 物理参数
```zen-c
DISC_RADIUS = 18       // 圆盘半径
FRICTION = 3           // 摩擦力 (像素/帧)
MIN_SPEED = 1          // 停止阈值
MAX_POWER = 25         // 最大发射力度
```

### 3.3 游戏状态机
```
Aiming → (mouse up) → Sliding → (all stopped) → RoundEnd → (next round) → Aiming
                                                              ↓
                                                         GameEnd
```

### 3.4 数据结构
```zen-c
struct Disc {
    x, y: int          // 位置
    vx, vy: int        // 速度
    owner: int         // 0=玩家1, 1=玩家2
    active: bool       // 是否在滑行
    scored: bool       // 是否已计分
}

enum Phase {
    Aiming,     // 瞄准中
    Sliding,    // 滑动中
    RoundEnd,   // 回合结束
    GameEnd     // 游戏结束
}
```

## 5. 控制方案

### 5.1 鼠标控制
1. 在发射线附近按住鼠标左键开始瞄准
2. 拖拽鼠标调整方向和力度（向后拖=向前发射）
3. 松开鼠标发射圆盘
4. 拖拽距离决定力度（最大 MAX_POWER）
5. 拖拽角度决定发射方向

### 5.2 键盘控制
| 按键 | 功能 |
|------|------|
| ESC | 退出游戏 |

## 6. 性能优化

- 帧缓冲直接写入，无中间缓冲区
- 圆形绘制使用 Bresenham 算法
- 仅在状态变化时重绘
- 固定 60 FPS 帧率限制
