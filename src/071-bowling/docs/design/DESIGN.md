# 保龄球游戏设计文档

## 1. 概述

- **项目名称**: 071-Bowling (保龄球)
- **游戏类型**: 俯视角度瓶球道保龄球
- **核心玩法**: 鼠标瞄准+拖拽掷球，模拟真实保龄球物理
- **目标平台**: Windows

## 2. 游戏规则

### 经典十瓶规则
- 10个瓶呈三角形排列
- 每局10格，每格最多2投
- 每投可多次击球直到所有球静止
- 第10格全中可获第3投

### 计分规则
- Strike (X): 10格全中，10分+后续两球
- Spare (/): 补中，10分+后续一球
- 普通: 实际击倒数

## 3. 技术设计

### 框架
- Zen-C + 003-framework
- 俯视角度二维渲染
- 整数像素运算

### 物理系统
- **球运动**: 速度向量 + 摩擦衰减
- **球瓶碰撞**: 圆形碰撞检测 + 动量传递
- **瓶瓶碰撞**: 连锁反应模拟
- **沟槽检测**: 边界检测

### 渲染层次
1. 背景 (深色)
2. 标题和分数板
3. 沟槽
4. 球道 (木纹)
5. 瓶区
6. 球
7. 瞄准线和力度条
8. 状态文字

## 4. 数据结构

### Pin 结构
```
x, y: 瓶心坐标
vx, vy: 速度向量
standing: 是否站立
fallen: 是否已倒
radius: 半径
color: 颜色
number: 瓶号 (1-10)
fallenTime: 倒下持续时间
```

### Ball 结构
```
x, y: 球心坐标
vx, vy: 速度向量
radius: 半径
rolling: 是否在滚动
inGutter: 是否在沟槽
```

### BowlingState 结构
```
ball: 保龄球
pins: 瓶数组 (10个)
pinCount: 瓶数量
status: 游戏状态
currentFrame: 当前Frame (0-9)
currentRoll: 当前投球次数 (1-3)
totalScore: 总分
frames: 10个Frame记录
```

### Frame 结构
```
firstRoll: 第1投击倒数
secondRoll: 第2投击倒数
pinsDown: 本Frame总击倒数
score: 本Frame得分
```

## 5. 游戏状态机

```
Ready → (点击拖拽) → Aiming
Aiming → (松开) → Throwing
Throwing → (球停+瓶停) → Ready 或 FrameDone
GameOver → (点击) → Ready (重新开始)
```

## 6. 瓶布局

```
    1        <- 第1排1瓶 (顶)
   2 3       <- 第2排2瓶
  4 5 6      <- 第3排3瓶
 7 8 9 10    <- 第4排4瓶 (底)
```

## 7. 控制说明

| 操作 | 功能 |
|------|------|
| 点击+向上拖拽 | 瞄准+蓄力 |
| 松开 | 掷球 |
| ESC | 退出游戏 |

## 8. 球道尺寸

- 游戏窗口: 500 x 700 像素
- 球道: 300 x 540 像素
- 瓶半径: 8 像素
- 球半径: 14 像素
- 沟槽宽度: 30 像素
