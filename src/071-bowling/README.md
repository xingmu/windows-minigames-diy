# 071-Bowling - 保龄球游戏 🧘

经典十瓶保龄球游戏，使用 Zen-C + 003-framework 开发。

## 🎮 游戏特性

- **经典十瓶**: 标准保龄球三角形瓶阵
- **真实物理**: 球体碰撞、瓶体倒下、摩擦减速
- **鼠标控制**: 点击拖拽蓄力，向上滑动掷球
- **力度显示**: 实时蓄力条，红黄绿表示力度
- **专业计分**: 标准保龄球计分规则，支持 Strike/Spare

## 🎯 游戏规则

### 计分规则
| 情况 | 符号 | 计分 |
|------|------|------|
| Strike (全中) | X | 10分 + 后续两球 |
| Spare (补中) | / | 10分 + 后续一球 |
| 普通 | 数字 | 实际击倒数 |

### 回合说明
- 每局10格，每格最多2投
- 第10格全中可获第3投
- 最多12投决定胜负

## 🔧 构建说明

### Windows 构建

```bash
# 使用批处理脚本
build_windows.bat

# 或使用 Makefile
make windows
```

### Linux/macOS 构建

```bash
make
```

### 运行游戏

```bash
make run
```

## 🛠️ 技术栈

- **语言**: Zen-C
- **框架**: 003-framework
- **平台**: Windows (Win32 API)
- **渲染**: 帧缓冲直接写入

## 📝 游戏状态

- **Ready**: 等待掷球
- **Aiming**: 瞄准蓄力中
- **Throwing**: 球运动中
- **Counting**: 计数中
- **GameOver**: 游戏结束
