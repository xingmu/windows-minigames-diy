# 089-Pingpong 乒乓球街机版 🏓

![Zen-C](https://img.shields.io/badge/Zen-C-游戏-blue)
![Platform](https://img.shields.io/badge/Platform-Windows-green)

## 游戏简介

089-Pingpong 是一款**街机增强版**的乒乓球双人对战游戏，在经典乒乓球基础上增加了道具系统、粒子特效和系列赛模式。

**与以往版本的区别：**
- **006-Pong**：玩家 vs 电脑AI
- **074-Pingpong**：玩家1 vs 玩家2（双人本地对战）
- **089-Pingpong（本作）**：街机增强版，带道具、技能、粒子特效和系列赛模式

## 游戏特色

- 🎮 **双人对战**：两名玩家在同一键盘上对战
- ⚡ **道具系统**：击球掉落道具，可获得各种增益效果
  - **S (红色)**：加速 - 球向对方飞得更快
  - **s (蓝色)**：减速 - 球向自己飞得更慢
  - **W (绿色)**：加宽挡板 - 挡板高度增加50%
  - **N (紫色)**：变窄挡板 - 挡板高度减少40%
  - **M (黄色)**：多球 - 激活额外的球
- ✨ **粒子特效**：击中、得分、获取道具时的视觉反馈
- 🔥 **连击系统**：记录连续击球次数，连击越高分数加成越多
- 🏆 **系列赛模式**：三局两胜制
- 💥 **屏幕震动**：强力击球时的震动效果

## 操作说明

### 玩家1 (左侧)
| 按键 | 功能 |
|------|------|
| W | 向上移动 |
| S | 向下移动 |

### 玩家2 (右侧)
| 按键 | 功能 |
|------|------|
| ↑ | 向上移动 |
| ↓ | 向下移动 |

### 通用
| 按键 | 功能 |
|------|------|
| 空格 | 发球 |
| R | 重新开始（游戏结束后）|
| ESC | 退出游戏 |

## 游戏规则

- **系列赛**：三局两胜制
- **单局**：先到 **11分** 的玩家获胜
- **道具**：球击中挡板后有概率掉落道具，捡到后自动使用
- **连击**：连续击球增加分数倍率

## 构建说明

### Windows

```bash

build_windows.bat

```

```bash
make run
```

### 清理构建

```bash
make clean
```
