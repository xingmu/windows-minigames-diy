# 018-Mahjong 设计文档

## 1. 项目概述

### 项目名称
Zen-C 麻将游戏 (018-Mahjong)

Windows 桌面游戏

使用 Zen-C 语言开发的经典麻将游戏，支持玩家与电脑AI对战，具备完整的吃、碰、杠、胡判定逻辑。

- Zen-C 语言学习者
- 复古游戏玩家

## 3. 麻将牌定义

### 牌的类型 (TileType)

```
万子 (Wan): Wan1-Wan9      [9种 x 4张 = 36张]
筒子 (Tong): Tong1-Tong9   [9种 x 4张 = 36张]
索子 (Suo): Suo1-Suo9     [9种 x 4张 = 36张]
风牌 (Feng): Dong,Nan,Xi,Bei [4种 x 4张 = 16张]
```

## 4. 游戏规则

### 座位和方向

```
        AI1 (上家)
         
AI3 (对家)    AI2 (下家)
         
        玩家
```

### 发牌规则
- 庄家先摸牌，共14张
- 其他三家各13张
- 每人轮流摸1张、打1张

### 操作规则

#### 吃 (Chi)
- 只能吃下家打出的牌
- 只能组成顺子（3张连续同花色）
- 例如：下家打出"5万"，手牌有"4万"和"6万"可吃

#### 碰 (Peng)
- 任意玩家打出牌，手牌中有2张相同可碰
- 碰后需打出一张牌

#### 杠 (Gang)
- 明杠：碰牌后，手牌有第4张相同
- 暗杠：手牌中有4张相同
- 加杠：碰后摸到第4张
- 杠后补摸一张牌

#### 胡 (Hu)
- 基本要求：4组面子 + 1对将
- 面子类型：顺子(3张连续)、刻子(3张相同)
- 自摸胡：自己摸到胡牌
- 点炮胡：别人打出胡牌

### 胡牌判定算法

```csharp
// 面子组成检测（递归）
function canFormMeld(counts):
    找到第一个有牌的索引 i
    
    if 没有牌了:
        return true
    
    // 尝试刻子
    if counts[i] >= 3:
        counts[i] -= 3
        if canFormMeld(counts):
            return true
        counts[i] += 3
    
    // 尝试顺子（数牌）
    if i是数牌 and i+1,i+2都有牌:
        counts[i]--, counts[i+1]--, counts[i+2]--
        if canFormMeld(counts):
            return true
    
    return false

// 胡牌检测
function canHu(hand):
    if 手牌数不是3n+2:
        return false
    
    // 枚举每个对子作为将牌
    for each tile with count >= 2:
        counts[tile] -= 2
        if canFormMeld(counts):
            return true
        counts[tile] += 2
    
    return false
```

## 5. AI 设计

### AI 难度级别

```csharp
enum AILevel {
    Easy,    // 简单：随机出牌
    Normal,  // 普通：避免放炮
    Hard,    // 困难：优化胡牌
}
```

### Easy AI
- 随机选择一张牌打出
- 不考虑任何策略

### Normal AI
- 优先打孤张（只出现1次的牌）
- 避免打危险牌（可能让别人胡的牌）
- 字牌优先打出

### Hard AI
- 计算每张牌的"价值"
- 保留搭子（组成顺子的潜力）
- 边张（1、9）和字牌价值低
- 中张（4、5、6）价值高
- 优先保留有对子的牌

```csharp
function aiHardDecision(hand):
    for each tile in hand:
        score = 0
        
        // 数牌关联计算
        if tile是数牌:
            score += 相邻牌数量 * 2
            score += 间邻牌数量
            if tile是中张(4-6):
                score += 3
            if tile是边张(1,9):
                score -= 2
        
        // 刻子加分
        if count[tile] >= 3:
            score += 10
        
        // 对子加分
        if count[tile] == 2:
            score += 5
        
        // 选择最高分
        选择 score 最高的牌打出
```

## 9. 游戏流程

### 开始阶段
1. 显示开始界面
2. 玩家点击"开始游戏"

### 发牌阶段
1. 洗牌
2. 掷骰子
3. 按顺序发牌（庄家14张，其他13张）

### 游戏阶段
```
while 牌墙有牌:
    for each player in [庄家, 下家, 对家, 上家]:
        1. 摸牌
        2. 检查杠
        3. 打出1张牌
        4. 其他玩家检查吃碰杠胡
        5. 如有人胡牌，游戏结束
```

### 结束条件
- 有人胡牌
- 牌墙摸完（荒局）
