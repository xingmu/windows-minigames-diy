# 扫雷增强版 (MinesweeperPlus) 设计文档

## 1. 项目概述

- **项目名称**: MinesweeperPlus (扫雷增强版)
- **项目编号**: 034
- **框架版本**: Zen-C Framework v1.0.0
- **目标平台**: Windows

## 2. 游戏特点

### 2.1 核心玩法
- 经典扫雷玩法
- 三个难度级别
- 首次点击安全保护
- 连锁翻开机制
- 旗帜标记系统
- 数字地毯（智能展开）

### 2.2 难度设置

| 难度 | 行列 | 地雷数 |
|------|------|--------|
| 初级 | 9×9 | 10 |
| 中级 | 16×16 | 40 |
| 高级 | 30×16 | 99 |

### 3.2 类设计

#### MineBoard 类
- `board`: 二维格子数组
- `firstClick`: 首次点击标志
- `gameOver`: 游戏结束标志
- `victory`: 胜利标志
- `reveal(row, col)`: 揭开格子
- `toggleFlag(row, col)`: 切换旗帜标记
- `chainReveal(row, col)`: 连锁揭开

#### Cell 结构
- `isMine`: 是否为地雷
- `state`: 格子状态 (Hidden/Revealed/Flagged/Question)
- `adjacentMines`: 周围地雷数

## 4. 操作控制

| 操作 | 按键/鼠标 | 功能 |
|------|----------|------|
| 揭开格子 | 左键 | 揭开指定位置的格子 |
| 标记地雷 | 右键 | 标记/取消地雷位置 |
| 智能展开 | 中键 | 当周围标记数等于数字时展开 |
| 重新开始 | F2 | 开始新游戏 |
| 初级难度 | 1 | 切换到初级难度 |
| 中级难度 | 2 | 切换到中级难度 |
| 高级难度 | 3 | 切换到高级难度 |

## 5. 算法设计

### 5.1 地雷布置
- 首次点击后布置地雷
- 以首次点击位置为中心3×3区域不放置地雷
- 使用线性同余随机数生成器

### 5.2 连锁翻开
- 当揭开的格子周围雷数为0时
- 递归揭开周围8个格子
- 继续检查揭开的格子是否也需要展开

### 5.3 数字地毯
- 当点击已揭开的数字格时
- 统计周围标记的旗帜数
- 如果旗帜数等于数字，则展开周围未标记的格子

## 6. 状态机

```
Ready -> Playing -> Won/Lost
         ^          |
         |          |
         +----F2----+
```

- Ready: 等待第一次点击
- Playing: 游戏进行中
- Won: 胜利（所有非雷格子已揭开）
- Lost: 失败（踩到地雷）

## 7. 视觉设计

### 7.1 颜色方案
- 背景色: RGB(192, 192, 192) 灰色
- 数字颜色: 1-蓝色, 2-绿色, 3-红色, 4-深蓝, 5-深红
- 地雷: 黑色
- 旗帜: 红色

### 7.2 格子大小
- 格子尺寸: 24×24 像素
- 格子间距: 2 像素
- 总计: 26×26 像素/格子
