# Memory 记忆翻牌游戏

Zen-C 语言开发的经典记忆翻牌游戏。

## 游戏规则

- **目标**: 在最少的步数内，找出所有成对的相同图案
- **操作**: 点击卡牌翻转，查看图案
- **胜利条件**: 找到所有配对

## 操作说明

| 操作 | 功能 |
|------|------|
| 鼠标点击 | 翻转卡牌 |
| R | 重新开始游戏 |
| ESC | 退出游戏 |

## 游戏特性

- 4×4 卡牌网格 (共 16 张 = 8 对)
- 8 种不同图案
- 彩色图案设计
- 分数和步数统计
- 翻牌动画效果

## 编译

### Windows

```batch
build_windows.bat
```

### Linux/macOS

```bash
make
```
