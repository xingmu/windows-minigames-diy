# Zen-C 中国象棋 (Chinese Chess)

Zen-C 语言实现的中国象棋游戏，玩家 vs 电脑，红先手。

## 🎮 游戏特性

- **完整棋盘**: 9x10 标准中国象棋棋盘
- **7种棋子**: 帅(将)、仕(士)、相(象)、马、车、炮、兵(卒)
- **正确规则**: 所有棋子移动规则正确实现
- **鼠标操作**: 点击选择棋子，点击移动
- **AI对手**: 简单但具有挑战性的电脑AI
- **视觉反馈**: 选中高亮、上一步标记

## 🕹️ 操作说明

- **左键点击**: 选择棋子或移动棋子
- **右键点击**: 取消当前选择
- **ESC键**: 退出游戏

## ♟️ 棋子规则

| 棋子 | 红方 | 黑方 | 移动规则 |
|------|------|------|----------|
| 将 | 帥 | 將 | 横竖走一步，不能出宫 |
| 仕 | 仕 | 士 | 斜线走一步，不能出宫 |
| 相 | 相 | 象 | 斜线走两步，不能过河 |
| 马 | 馬 | 馬 | L形走日，不能蹩马腿 |
| 车 | 車 | 車 | 横竖走任意步 |
| 炮 | 炮 | 炮 | 横竖走，吃子需隔山打牛 |
| 兵 | 兵 | 卒 | 过河前只前进，过河后可左右 |

## 🔧 构建说明

### Windows


```bash
make
```

### Linux/macOS

```bash
make
```
