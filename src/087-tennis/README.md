# 087-Tennis 网球游戏

俯视角双人对战网球游戏，玩家与电脑对战，体验真实的网球竞技。

## 🎮 游戏特色

- **俯视角网球场地**：从上方俯瞰整个球场，更直观地观察球路
- **真实物理效果**：重力、空气阻力、球的旋转（上下旋）
- **完整网球计分**：15-30-40计分制，局、盘、比赛三级胜利
- **球路轨迹预测**：显示球的运动轨迹，提前预判落点
- **智能AI对手**：AI会根据球的位置和速度预测落点，具备不同难度

## 🕹️ 操作说明

| 按键 | 功能 |
|------|------|
| ← → / A D | 左右移动球拍 |
| 空格键 | 挥拍 / 发球 |
| 鼠标左键 | 挥拍 / 发球 |
| R | 重新开始游戏 |

## 🏆 胜利条件

- **分（Point）**：先得4分且领先2分以上获胜
- **局（Game）**：先赢6局且领先2局以上获胜
- **盘（Set）**：先赢2盘获得比赛胜利

## 📦 构建说明

### 前置要求

需要安装 Zen-C 编译器 `zc`。

### Windows 构建

```bash
# 使用 Makefile
make

# 或使用构建脚本
build_windows.bat
```

### 运行游戏

```bash
make run
```

## 🎯 游戏规则

1. **发球**：每局开始由指定方发球，按空格或点击发球区发球
2. **换发球**：每次丢分后换对方发球
3. **出界**：球落到对方场地边界内即得分
4. **触网**：球触网后落在本方仍可继续
5. **Deuce**：40-40后需要连续得2分才能获胜

## 🎨 界面说明

- **绿色区域**：草地球场
- **棕色区域**：泥土发球区
- **白色线条**：球场边线、发球线、中线
- **黄色圆形**：网球
- **黄色轨迹**：球的飞行轨迹预测
