# 016-Billiards - 台球游戏 🧘

俯视图斯诺克台球游戏，使用 Zen-C + 003-framework 开发。

## 🎮 游戏特性

- **俯视斯诺克**: 经典斯诺克规则，15红球+6彩球+1白球
- **真实物理**: 球体碰撞、边界反弹、摩擦衰减
- **鼠标控制**: 点击拖拽瞄准，蓄力击球
- **力度显示**: 实时蓄力条，红黄绿表示力度

## 🎯 游戏规则

### 球与分值
| 球 | 颜色 | 分值 |
|---|------|------|
| 白球 | 象牙白 | - |
| 红球 | 红色 | 1分 |
| 黄球 | 黄色 | 2分 |
| 绿球 | 绿色 | 3分 |
| 棕球 | 棕色 | 4分 |
| 蓝球 | 蓝色 | 5分 |
| 粉球 | 粉色 | 6分 |
| 黑球 | 黑色 | 7分 |

### 控制说明
- **点击+拖拽**: 从白球拖出方向，松开击球
- **拖拽距离**: 决定击球力度
- **空格键**: 轻击
- **ESC**: 退出游戏

## 🔧 构建说明

### Windows 构建

```bash
# 使用批处理脚本
build_windows.bat

# 或使用 Makefile
make windows
```

### Linux/macOS 构建

```bash
make
```

### 运行游戏

```bash
make run
```

## 🛠️ 技术栈

- **语言**: Zen-C
- **框架**: 003-framework
- **平台**: Windows (Win32 API)
- **渲染**: 帧缓冲直接写入

## 📝 游戏状态

- **Ready**: 等待击球
- **Dragging**: 拖拽蓄力中
- **Moving**: 球运动中
- **GameOver**: 结束
