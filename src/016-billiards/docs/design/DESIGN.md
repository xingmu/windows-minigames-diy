# 台球游戏设计文档

## 1. 概述

- **项目名称**: 016-Billiards (台球)
- **游戏类型**: 俯视图斯诺克台球
- **核心玩法**: 鼠标瞄准+拖拽击球，模拟真实台球物理
- **目标平台**: Windows

## 2. 游戏规则

### 斯诺克基础规则
- 1个白色母球 (Cue Ball)
- 15个红色球 (1分/球)
- 6个彩色球:
  - 黄球 (2分)
  - 绿球 (3分)
  - 棕球 (4分)
  - 蓝球 (5分)
  - 粉球 (6分)
  - 黑球 (7分)

### 计分规则
- 落袋球按分值计分
- 白球落袋扣4分并重置

## 3. 技术设计

### 框架
- Zen-C + 003-framework
- 俯视图二维渲染
- 整数像素运算

### 物理系统
- **球体碰撞**: 圆形碰撞检测 + 动量守恒
- **边界碰撞**: 反射定律 (入射角=反射角)
- **摩擦力**: 每帧速度衰减
- **落袋检测**: 距离检测

### 渲染
- 帧缓冲直接写入
- 双缓冲防闪烁
- 层次: 台布 → 边框 → 袋口 → 球 → UI

### 球杆控制
- 鼠标位置确定瞄准方向
- 拖拽起点到终点计算力度和方向
- 力度条实时显示

## 4. 数据结构

### Ball 结构
```
x, y: 球心坐标
vx, vy: 速度向量
radius: 半径
color: 颜色
number: 球号 (0=白球, 1-21=彩球)
pocketed: 是否落袋
active: 是否在场
```

### BilliardsState 结构
```
balls: 球数组
ballCount: 球数量
cueBall: 白球引用
status: 游戏状态
score: 当前分数
shots: 击球次数
mouseX/Y: 鼠标坐标
power: 蓄力值
aimX/Y: 瞄准方向
```

## 5. 游戏状态机

```
Ready → (点击拖拽) → Dragging
Dragging → (松开) → Moving
Moving → (所有球静止) → Ready
Any → (犯规) → Ready (扣分重置白球)
```

## 6. 关键算法

### 球球碰撞
1. 计算两球圆心距离
2. 距离 < r1+r2 则碰撞
3. 计算碰撞法向量
4. 根据动量守恒计算新速度
5. 分离重叠球体

### 瞄准击球
1. 鼠标位置 - 白球位置 = 瞄准向量
2. 拖拽距离 / 5 = 蓄力值
3. 松开时沿瞄准方向发射白球
