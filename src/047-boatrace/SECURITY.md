# 安全策略 - Zen-C BoatRace Game

## 📋 概述

本项目是一个使用 Zen-C 语言开发的 Windows 桌面游戏应用。

## 🔒 安全原则

1. **最小权限**：游戏只需要基本的窗口和图形绘制权限
2. **无网络访问**：游戏不需要网络连接
3. **本地存储**：所有数据存储在本地，无云端同步
4. **无外部依赖**：不加载任何外部未签名代码

## ⚠️ 已知限制

- 游戏为单机桌面应用，不处理任何网络数据
- 不存储用户敏感信息
- 不使用任何第三方追踪或分析服务

## 🛡️ 建议

1. 从官方仓库下载游戏可执行文件
2. 保持操作系统和编译器更新
3. 使用杀毒软件扫描未知来源的可执行文件

## 📞 报告问题

如果您发现任何安全相关问题，请通过 GitHub Issues 报告。
