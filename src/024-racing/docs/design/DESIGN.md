# 024-Racing 设计文档

## 概述

**游戏名称**：024-Racing  
**类型**：俯视角赛车游戏  
**引擎**：Zen-C + 003-framework v1.0.0  
**版本**：1.0.0

## 游戏设计

### 核心玩法

玩家驾驶赛车在矩形赛道上竞速，通过：
- ↑ 键加速
- ← → 键转向
- 在赛道上行驶获得正常速度
- 进入草地会减速

### 物理模型

```
赛车状态:
- position (x, y)
- angle (朝向角度)
- speed (当前速度)

加速: speed += acceleration * dt
摩擦: speed *= friction
转向: angle += turn_speed * (speed / max_speed) * dt
移动: x += cos(angle) * speed
      y += sin(angle) * speed
```

```
│      ║                      ║      │

│ 草地    ═══════════════════  草地 │

```

### 检查点系统

赛道分为 4 个检查点区域（上、右、下、左），赛车经过每个检查点推进一圈进度。
