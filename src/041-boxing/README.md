# 041-Boxing（拳击游戏）👊

Zen-C + 003-framework 实现的拳皇风格拳击游戏。

## 🎮 游戏特色

- **拳皇风格连招系统** - 方向键组合键出拳
- **双人对战** - 玩家 vs CPU
- **血条系统** - 实时显示双方血量
- **连击计数** - 打出精彩Combo
- **华丽特效** - 击中时的视觉反馈

## 🎮 操作说明

### 玩家1 (左侧)

| 按键 | 动作 |
|------|------|
| ← + A | 左直拳 |
| → + A | 右直拳 |
| ↑ + A | 上勾拳 |
| ↓ + A | 下扫拳 |
| A + A | 二连击 |
| ← + B | 左摆拳 |
| → + B | 右摆拳 |
| A + B | 组合拳 |
| ← + C | 防御 |
| → + C | 前进 |
| C + C | 特殊技 |

### 玩家2 (右侧)

| 按键 | 动作 |
|------|------|
| J | 左直拳 |
| L | 右直拳 |
| I | 上勾拳 |
| K | 下扫拳 |
| J + J | 二连击 |
| U | 左摆拳 |
| O | 右摆拳 |
| J + U | 组合拳 |
| ← + K | 防御 |
| → + K | 前进 |
| K + K | 特殊技 |

## 🕹️ 游戏规则

1. 双方各有 100 点血量
2. 使用连招可以造成更高伤害
3. 防御可以减少受到的伤害
4. 血量归零者失败
5. 先赢 3 局者获胜
