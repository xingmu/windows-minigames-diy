# 046-Shuffleboard 沙狐球

Zen-C 编写的沙狐球游戏 - Shuffleboard

## 游戏简介

沙狐球是一种在长直滑道上滑动金属圆盘进行计分的游戏。本游戏模拟经典的沙狐球玩法，支持双人对战。

## 游戏规则

1. 两名玩家轮流发射沙狐球
2. 每回合每人有一次发射机会
3. 球停在计分区内即可获得相应分数
4. 球出界(越过顶线)则不得分
5. 球可以撞击其他球使其移动
6. 5回合后分数高的玩家获胜

## 计分区域

```
┌─────────────────────────────┐
│  10分 (金色)                │  ← 最近/最难
│  8分  (银色)                │
│  7分  (青铜)                │
│  6分  (深红)                │
│  5分  (蓝)                  │
│  4分  (绿)                  │
│  3分  (粉)                  │
│  2分  (紫)                  │
│  1分  (米色)                │  ← 最远/最容易
│  ─────────────────────────  │  ← 发射线
│  [  发射区  ]               │
└─────────────────────────────┘
```

## 操作说明

| 操作 | 按键/鼠标 |
|------|----------|
| 蓄力发射 | 鼠标点击并向上拖拽 |
| 发射 | 松开鼠标 |
| 跳过回合 | 空格键 |
| 退出游戏 | ESC |

## 蓄力技巧

- 向上拖拽距离越大，发射力度越大
- 力度条颜色提示:
  - 绿色: 力度适中
  - 黄色: 力度较大
  - 红色: 力度最大(容易出界)
