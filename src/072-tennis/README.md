# 🎾 072-Tennis - 网球游戏

基于 Zen-C + 003-framework 开发的网球游戏。

## 🎮 游戏特色

- **真实网球规则**: 15-30-40 计分系统，先赢6局并领先2局获胜
- **发球机制**: 双人对抗，需要发球开始
- **智能AI**: 电脑对手具有不同的反应速度和失误率
- **视觉效果**: 绿色球场、白色边线、黄色网球、球拍击球特效

## 🕹️ 操作说明

| 按键 | 功能 |
|------|------|
| ↑ / W | 向上移动球拍 |
| ↓ / S | 向下移动球拍 |
| 空格 | 开始发球 |
| R | 重新开始 |

## 📂 项目结构

```
072-tennis/
├── src/
│   ├── platform_api.zc         # 统一平台接口
│   ├── platform_api_windows.zc # Windows 平台实现
│   ├── game.zc                # 游戏框架核心
│   ├── game_logic.zc          # 网球游戏逻辑
│   └── main.zc                # 入口点
└── README.md                   # 本文件
```

## 🎯 游戏规则

1. **计分系统**: 每分 0→15→30→40→Game
2. **局制**: 先赢6局且领先2局赢得一盘
3. **比赛**: 先赢2盘获胜
4. **换发球**: 每分结束后换发球

## 🔧 构建

使用 Zen-C 编译器构建:

```bash
zcc src/main.zc -o tennis.exe
```
