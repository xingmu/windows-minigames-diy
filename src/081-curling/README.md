# 081-Curling 冰壶游戏

Zen-C 语言开发的冰壶游戏，使用 003-framework 框架。

## 游戏简介

冰壶是一项源自苏格兰的冬季运动，两人或两队对抗，在冰面上滑行冰壶石，使石块停在目标区域得分。

## 游戏规则

### 基本规则
- 每局每队各有 8 次投掷机会
- 共 8 局比赛
- 红队先手投掷
- 投掷后可通过"清扫"减少冰面摩擦力，帮助石头滑得更远

### 得分规则
- 只有最接近圆心的队伍得分
- 得分 = 该队所有在圆垒(house)内的石头数量
- 8局结束后，总分高的队伍获胜

### 投掷控制
- 点击并向上拖拽蓄力
- 拖拽左右偏移控制瞄准角度
- 拖拽方向也控制石头的旋转方向
- 松开鼠标投掷

### 清扫机制
- 石头滑行时，按住鼠标左键进行清扫
- 清扫可以减少冰面摩擦力，让石头滑得更直更远

## 操作说明

| 操作 | 功能 |
|------|------|
| 点击+拖拽 | 蓄力投掷 |
| 滑行时按住 | 清扫(减少摩擦) |
| R 键 | 重新开始 |
| ESC | 退出游戏 |

## 技术特性

- 冰面 curling 物理效应 (旋转导致弧线)
- 石头间碰撞模拟
- 清扫机制减少摩擦力
- 8局计分系统

## 编译运行

### Linux/macOS
```bash
make
./build/curling
```

### Windows
```bash
make windows
build\curling.exe
```

### 快速运行
```bash
make run
```

## 开发说明

使用 Zen-C 语言开发，基于 003-framework 游戏框架。

### 游戏常量
- 冰道尺寸: 500x750 像素
- 石头半径: 16 像素
- 冰面摩擦: 1
- 最大速度: 20
