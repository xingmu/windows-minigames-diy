# Tower Defense - Zen-C

塔防游戏 - 使用 Zen-C 语言开发的经典塔防游戏。

## 游戏介绍

敌人从路径上涌来，你需要建造防御塔来消灭它们，保护你的基地。

## 游戏特性

- **3种防御塔类型**:
  - 箭塔 (Arrow): 快速攻击，攻击力低，射程远
  - 炮塔 (Cannon): 慢速攻击，攻击力高，范围伤害
  - 冰塔 (Ice): 减速敌人，攻击力低

- **4种敌人类型**:
  - 普通敌人 (Basic): 标准属性
  - 快速敌人 (Fast): 速度快但血量低
  - 坦克敌人 (Tank): 血量高但速度慢
  - Boss敌人 (Boss): 出现在第5波及之后，极高血量

- **10个波次**: 逐步增加难度
- **金币系统**: 击杀敌人获得金币，购买防御塔
- **建造系统**: 在空地上建造防御塔

## 操作说明

| 按键 | 功能 |
|------|------|
| 空格 (Space) | 开始游戏 / 进入下一波 |
| 1 | 选择箭塔 |
| 2 | 选择炮塔 |
| 3 | 选择冰塔 |
| 鼠标左键 | 建造防御塔 |
| ESC | 退出游戏 |

## 游戏规则

1. 游戏开始时有 200 金币和 20 条生命
2. 点击空地建造防御塔
3. 敌人到达终点会扣除生命
4. 击杀敌人获得金币奖励
5. 击败 10 个波次即可获胜

## 防御塔信息

| 塔类型 | 价格 | 伤害 | 射程 | 攻击间隔 |
|--------|------|------|------|----------|
| 箭塔 | 50 | 10 | 120 | 500ms |
| 炮塔 | 100 | 30 | 100 | 1200ms |
| 冰塔 | 75 | 5 | 100 | 800ms |

## 构建

### Windows

```bash

```

```bash

```

### 运行

```bash
./build/towerdefense
```
