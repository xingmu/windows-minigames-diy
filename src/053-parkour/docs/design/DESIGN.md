# Parkour 游戏设计文档

## 概述

Parkour（跑酷）是一款经典横版无尽跑酷游戏，玩家控制角色自动向右奔跑，通过跳跃和二段跳躲避障碍物、收集金币。

## 游戏机制

### 核心循环

1. 游戏开始 → 世界自动向右滚动
2. 玩家通过空格/上箭头跳跃
3. 空中可再次按跳跃键进行二段跳
4. 躲避尖刺障碍物
5. 收集空中或地面的金币
6. 到达 10000 米处胜利

### 物理系统

- **重力加速度**: 1000 px/s²
- **跳跃初速度**: -450 px/s（第一跳）
- **二段跳初速度**: -400 px/s
- **世界滚动速度**: 200 px/s（起始），最高 400 px/s
- **速度递增**: 每 10 秒增加 5 px/s

### 障碍物生成

- 尖刺：间隔 180-350 像素生成
- 金币：间隔 80-200 像素生成
- 悬浮平台：间隔 100-250 像素生成，高度 180-270 像素

### 碰撞检测

- AABB 矩形碰撞
- 玩家碰撞箱比视觉尺寸小 4 像素（左右各 2 像素）
- 平台碰撞仅从上方生效（掉落检测）

## 视觉设计

### 色彩方案

- 天空：深蓝渐变 (#141E50 → #3C64B8)
- 建筑（远景）：深蓝灰 (#232F46)
- 建筑（近景）：蓝灰 (#32374B)
- 地面：深棕 (#3C3228)
- 玩家：青色上衣 (#3CB4DC)，蓝色裤子 (#2828C8)
- 金币：金色 (#FFD200)
- 尖刺：银灰 (#B4B4B4)

### UI 设计

- HUD 顶部固定显示：分数、生命、距离、金币数量、速度
- 开始画面：游戏标题 + 操作说明
- 结束画面：最终分数 + 距离
- 胜利画面：祝贺信息 + 统计

## 粒子系统

- 跳跃尘埃：地面接触时生成
- 金币收集：爆发式粒子
- 受伤爆炸：红色危险色粒子
- 二段跳：金色小粒子

## 性能目标

- 60 FPS 稳定运行
- 对象池复用，减少 GC
- 屏幕外对象自动清理
